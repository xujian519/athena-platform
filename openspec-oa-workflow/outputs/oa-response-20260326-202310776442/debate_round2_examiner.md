# 辩论记录 - 第2轮（完整）

## 🔍 审查员质疑

**核心质疑**：原则性规定的援引无法替代明确的规则性依据，且"疏忽"抗辩将动摇同日双申制度的运行基础。

### 三大质疑

**1. 原则与规则之辩**
- 《立法法》等都是**原则性条款**
- 《专利法》第9条、《实施细则》第41条是**明确规则性规定**
- 应优先适用明确规则，而非援引原则架空制度

**2. 禁止重复授权的刚性约束**
- 一发明一专利是基本原则
- 如果允许"疏忽"规避，禁止重复授权将形同虚设
- 制度设计要求申请人主动履行义务，非事后补救

**3. 程序信赖与可预期性**
- 审查员依法履职，撤销将损害公信力
- 申请人未在法定期限内行使权利，已产生失权效果
- "疏忽"理由将破坏期限制度严肃性

### 评价

**优点**：法律援引层次丰富，理论功底深厚
**不足**：
- 停留在抽象原则，未回应为何可例外适用
- 未触及"禁止重复授权"核心法理
- "疏忽"抗辩缺乏客观标准

### 结论

❌ **不接受**

理由：
1. 缺乏明确规则性依据
2. "疏忽"抗辩将动摇制度基础
3. 申请人应承担不利后果
4. 应通过立法/修法途径完善
