📝 Cypher查询示例

在Neo4j Browser中执行以下查询：

// 查看50个节点及其关系
MATCH (n)-[r]->(m)
RETURN n,r,m
LIMIT 50 // 查看实体关系
MATCH (e:Entity {type: "PATENT_NUMBER"})-[r:RELATION]->(m)
RETURN e,r,m
LIMIT 50 // 查看OpenClaw关系
MATCH (n:OpenClawNode)-[r:RELATED_TO|CITES]->(m:OpenClawNode)
RETURN n.title, type(r), m.title
LIMIT 50
