# 03-山药碎汤



## 一、原材料

1.1，山药碎；

1.2，胡萝卜丁；

1.3，香菇丁；

1.4，菜碎；

1.5，葱姜蒜；

1.6，味极鲜；

1.7，蚝油；

1.8，盐、味精（或鸡精）；

1.9，胡椒粉；

1.10，鸡蛋液。



## 二，步骤

2.1，山药用到刮碎，颗粒大时剁一下，备用；

2.2，胡萝卜切丁、香菇切丁备用；

2.3，选绿叶菜（如油菜、菠菜等）切碎备用；

2.4，准备葱姜蒜；

2.5，热锅冷油、依次下入胡萝卜丁、香菇丁、葱姜蒜煸炒，炒香；

2.6，加纯净水，烧开，加入山药碎；

2.7，加入盐、味精、蚝油、味极鲜、胡椒粉调味；煮约5分钟；

2.8，加入菜碎，煮1分钟；

2.9，加入鸡蛋液、芫荽出锅即得。
