## 🚀 快速部署

### 方式1：直接Python调用

```python
# 最简单的使用方式
from xiaonuo_unified_interface import XiaonuoUnifiedInterface, NLPRequest

# 创建接口
interface = XiaonuoUnifiedInterface()

# 处理请求
request = NLPRequest(text="帮我搜索机器学习教程")
response = interface.process_single(request)

print(f"意图: {response.intent}")
print(f"工具: {response.selected_tools}")
print(f"置信度: {response.confidence}")
```

### 方式2：使用NLP服务

```python
# 使用封装的NLP服务
from xiaonuo_nlp_service import XiaonuoNLPService

# 创建服务
service = XiaonuoNLPService()

# 处理文本
response = service.process_text(
    text="帮我搜索机器学习教程",
    user_id="user_001",
    session_id="session_001"
)

print(f"成功: {response.success}")
print(f"回复: {response.response_text}")
print(f"建议: {response.suggestions}")
```

### 方式3：HTTP API服务

```python
# 启动HTTP服务器
python xiaonuo_nlp_api.py --host 0.0.0.0 --port 8080

# 客户端调用
import requests

# 处理文本
response = requests.post('http://localhost:8080/process', json={
    'text': '帮我搜索机器学习教程',
    'user_id': 'user_001',
    'session_id': 'session_001'
})

result = response.json()
print(result)
```

## 🔧 部署模式

### 1. 单机部署

```bash
# 直接运行
python xiaonuo_nlp_service.py --mode standalone

# 或者作为后台服务
python xiaonuo_nlp_service.py --mode background
```

### 2. API服务部署

```bash
# 启动HTTP API
python xiaonuo_nlp_api.py --host 0.0.0.0 --port 8080

# 使用nginx反向代理
upstream nlp_backend {
    server localhost:8080;
}

server {
    listen 80;
    server_name your-domain.com;

    location /api/nlp {
        proxy_pass http://nlp_backend;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }
}
```

### 3. 容器化部署

```dockerfile
# Dockerfile
FROM python:3.9-slim

WORKDIR /app
COPY . /app/

RUN pip install -r requirements.txt

EXPOSE 8080

CMD ["python", "xiaonuo_nlp_api.py", "--host", "0.0.0.0", "--port", "8080"]
```

```bash

docker build -t xiaonuo-nlp .
docker run -p 8080:8080 xiaonuo-nlp
```

## 🔌 集成示例

### 1. 小诺智能体集成

```python
class XiaonuoAgent:
    def __init__(self):
        # 初始化NLP服务
        self.nlp_service = XiaonuoNLPService()

    def process_user_input(self, text, user_id=None, session_id=None):
        # NLP处理
        nlp_result = self.nlp_service.process_text(
            text=text,
            user_id=user_id,
            session_id=session_id
        )

        # 根据意图执行相应动作
        if nlp_result.intent == "query":
            return self._handle_query(nlp_result)
        elif nlp_result.intent == "recommend":
            return self._handle_recommendation(nlp_result)
        # ... 其他意图

    def _handle_query(self, nlp_result):
        # 执行查询逻辑
        tools = nlp_result.selected_tools
        params = nlp_result.parameters

        if "web_search" in tools:
            return self._web_search(params)
        elif "knowledge_graph" in tools:
            return self._query_knowledge_graph(params)
```

### 2. 微服务架构集成

```python
# 在其他微服务中调用NLP
import requests

class TaskService:
    def __init__(self):
        self.nlp_api_url = "http://nlp-service:8080"

    def process_task_request(self, text, user_id):
        # 调用NLP API
        response = requests.post(
            f"{self.nlp_api_url}/process",
            json={
                "text": text,
                "user_id": user_id
            }
        )

        nlp_result = response.json()

        # 使用NLP结果
        return self._create_task(nlp_result)
```

### 2. 常见问题

**Q: 意图识别准确率不理想**
A: 检查训练数据是否充足，考虑添加更多样例

**Q: 响应时间较长**
A: 调整缓存配置，增加max_workers

**Q: 内存占用过高**
A: 调整max_memory_mb配置，减少缓存大小

### 3. 优化建议
1. 定期清理过期会话
2. 监控缓存命中率
3. 根据实际负载调整配置
4. 定期更新训练模型
