## 一、核心判断标准

### 1.1 新产品用途发明

> 用途不能从结构或组成相似的已知产品预见到

### 1.2 已知产品用途发明

> 新用途不能从产品本身的结构、组成、已知性质及现有用途显而易见得出，而是**利用了新发现的性质并产生预料不到的技术效果**

### 1.3 判断核心

用途发明创造性判断的核心在于：
1. 考察**新用途与已知用途之间的关联性**
2. 判断新用途是否利用了**新发现的性质**（而非已知性质的简单延伸）
3. 是否产生了**预料不到的技术效果**

---

## 二、关键区分："新发现的性质" vs "已知性质的延伸"

### 2.1 不具备创造性的情形

**新旧用途共享同一机制 → 自然扩展 → 无创造性**

典型案例：第78285号复审决定
- 副干酪乳杆菌从防治变形链球菌龋扩展到防治远缘链球菌龋
- 新旧适应症致病机理高度关联：同一抗原、同群细菌、蛋白高度保守
- 技术人员有理由将同种细菌用于防治群内其他致龋菌
- 结论：同一机制的自然扩展，不具备创造性

### 2.2 具备创造性的情形

**新旧用途在技术原理或操作方式上存在根本区别 → 非自然扩展 → 有创造性**

判断要点：
1. 检测原理是否不同（如可见光 vs 近红外荧光）
2. 检测设备是否不同（如肉眼观察 vs 仪器探测）
3. 临床操作方式是否不同（如切开皮肤 vs 体表探测）
4. 所利用的产品性质是否为新发现的性质

## 三、反向教导在用途发明中的适用

### 3.1 双重反向教导论证框架

当现有技术满足以下条件时，构成反向教导：

**第一重：现有技术对"新性质"给出负面定性**
- 如：现有技术将荧光定性为"干扰因素"或"妨碍使用"
- 标准：(2020)最高法知行终185号——"明确排除该区别技术特征的应用"

**第二重：现有技术对"关键机制"给出负面定性**
- 如：现有技术将蛋白结合定性为"微弱结合"
- 进一步强化本领域技术人员不会朝该方向探索的判断

### 3.2 整体分析原则

依据(2019)最高法行再268号：
- 反向教导判断须从现有技术的**整体语境**分析
- 不能断章取义，截取"产生荧光"而忽略"妨碍使用"
- 反向教导认定要件：(1)公开相反特征；(2)明确指为缺陷

### 3.3 实务论证结构

```
已知产品新用途创造性论证模板：

第一层：区别特征和新性质
  → 新用途利用了何种新发现的性质？
  → 该性质与已知性质有何根本区别？

第二层：反向教导
  → 现有技术对该新性质给出何种定性？
  → 是否明确排除、明确指为缺陷？
  → 本领域技术人员的正常反应是回避还是利用？

第三层：预料不到的技术效果
  → 新用途在技术原理上有何质变？
  → 效果是否事先不可预测？

第四层：辅助因素（可选）
  → 是否解决了长期技术难题？
  → 是否克服了技术偏见？
```

## 四、典型案例分析：异硫蓝注射液荧光示踪剂（202411856719.5）

### 4.1 案件概况

- **申请人**: 山东大学齐鲁医院（杨其峰教授）
- **发明**: 异硫蓝注射液作为荧光示踪剂的应用
- **已知用途**: 蓝染料示踪（可见光肉眼观察）
- **新用途**: 荧光示踪（近红外仪器探测）
- **对比文件1**: CN106267240A——异硫蓝注射液制剂配方
- **对比文件2**: Santa Cruz等——蓝染料前哨淋巴结定位研究

### 4.2 创造性论证要点

**区别特征和新性质：**
- D2所述已知性质：异硫蓝"产生强烈荧光"（笼统物理现象）
- 本申请新发现性质：异硫蓝与血清蛋白结合后产生特定波段荧光光谱特性（620-690nm激发→690-790nm荧光）
- 两者为不同性质，非已知性质的简单延伸

**反向教导（第一重——D2）：**
- D2完整语境："produce strong fluorescence that **precludes use of** some fluorescent optical imaging agents"
- 异硫蓝荧光被定性为**妨碍其他荧光显像剂使用的干扰因素**
- 审查员截取前半句"产生强烈荧光"忽略后半句"妨碍了……的使用"，属于断章取义

**反向教导（第二重——D1[0013]）：**
- D1第[0013]段记载：异硫蓝"**与间质蛋白微弱结合**"
- 将蛋白结合定性为"微弱"，教导不重视蛋白结合效应
- 本申请恰恰利用了与血清蛋白（间质蛋白的主要成分）结合后的荧光特性

**预料不到的技术效果：**

| 对比维度 | D1/D2蓝染示踪 | 本申请荧光示踪 |
|---------|-------------|-------------|
| 检测原理 | 可见光肉眼观察 | 近红外荧光仪器探测 |
| 手术操作 | 必须切开皮肤 | 不切开皮肤即可体表探测 |
| 灵敏度 | 受组织深度和血液污染影响 | 可穿透组织，不受血液污染 |

从"切开皮肤肉眼观察蓝染"到"体表仪器探测荧光"——这是检测原理的**质变**，而非量变。

**科学机理支撑：**
- Tellier等(2012, PMID:23024922)首次报道PBV（三苯甲烷类染料）与HSA结合后量子产率提升32倍
- 游离态因"极快振动去激发"荧光极弱；蛋白结合限制分子内旋转→荧光增强
- 异硫蓝同属三苯甲烷类，具有相同机理

### 4.3 法律依据

1. 审查指南第十章§5.4：已知产品新用途，利用新发现性质+预料不到效果
2. 审查指南第四章§5.3：技术效果的质变
3. (2020)最高法知行终185号：反向教导→缺乏改进动机→有创造性
4. (2019)最高法行再268号：反向教导整体分析
5. 第78285号复审决定（反面参照）：机制相同=无创造性，本案机制不同=有创造性

## 五、实务建议

### 5.1 说明书记载要点

在撰写用途发明专利说明书时，应着重记载：
1. 新用途的**发现过程**
2. 新用途与已知性质的**区别**
3. 所产生的**意外技术效果**
4. 关键机理说明（如蛋白结合后的荧光增强机制）

### 5.2 复审/无效应对策略

1. **明确新旧性质的区别**：不能笼统地说"利用了X性质"，要具体说明是何种新的具体性质
2. **全面引用对比文件**：不能截取只言片语，要展示完整语境中的反向教导
3. **量化技术效果**：尽可能提供对比数据证明效果的质变或量变
4. **利用辅助判断因素**：同时主张长期技术难题、技术偏见等辅助因素
5. **区分机制**：明确新用途与已知用途在技术原理、检测设备、操作方式上的根本区别
