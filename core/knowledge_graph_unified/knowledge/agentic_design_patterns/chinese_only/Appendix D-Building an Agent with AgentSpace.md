# Appendix D-Building an Agent with AgentSpace** | <mark>附录 D

**Agentspace 现已更名为 Gemini Enterprise;考虑与原文保持一致的原则，本译文仍继续采用 AgentSpace 名称。

## Overview | <mark>概览</mark>

是一个旨在通过将人工智能融入日常工作流程来促进智能体驱动型企业发展的平台其核心是提供统一的搜索功能， 涵盖组织的全部数字足迹， 包括文档电子邮件和数据库该系统利用先进的人工智能模型（例如）来理解和整合来自不同来源的信息

该平台支持创建和部署专门的人工智能体， 这些智能体可以执行复杂任务并实现流程自动化这些智能体不仅仅是聊天机器人； 它们可以自主推理规划和执行多步骤操作例如， 智能体可以研究某个主题， 编写包含引文的报告， 甚至生成音频摘要

为了实现这一点， 构建了一个企业知识图谱， 映射了人员文档和数据之间的关系这使得人工智能够理解上下文并提供更相关更个性化的结果该平台还包含一个名为的无代码界面， 无需深厚的技术专业知识即可创建自定义智能体

此外， 支持多智能体系统， 不同的智能体可以通过协议的开放协议进行通信和协作这种互操作性支持更复杂更协调的工作流程安全性是其基础组件， 它还提供基于角色的访问控制和数据加密等功能来保护敏感的企业信息的最终目标是通过将智能自主系统直接嵌入到组织的运营结构中来提高生产力和决策能力

## Conclusion | <mark>结语</mark>

总而言之， 提供用于在组织现有的数字基础架构内开发和部署智能体的功能框架该系统的架构将复杂的后端流程（例如自主推理和企业知识图谱映射）链接到用于构建智能体的图形用户界面通过该界面， 用户可以通过集成各种数据服务并通过提示定义其操作参数来配置智能体， 从而构建定制的情境感知的自动化系统

这种方法抽象了底层的技术复杂性， 无需深厚的编程专业知识即可构建专用的多智能体系统其主要目标是将自动化分析和操作功能直接嵌入到工作流中， 从而提高流程效率并增强数据驱动的分析能力在实践教学方面， 我们提供动手学习模块， 例如上的实验， 它为技能学习提供结构化的环境
