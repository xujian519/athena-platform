Antonio Gulli 《Agentic Design Patterns: A Hands-On Guide to Building Intelligent Systems》****。 (Agent) 。

### 💻 | Running Code Examples

`codes/` 。

#### 1. | Environment Setup

```bash
#
git clone https://github.com/ginobefun/agentic-design-patterns-cn.git
cd agentic-design-patterns-cn

#
python3 -m venv venv
source venv/bin/activate  # macOS/Linux

#
pip install langchain langchain-community langchain-openai langgraph
```

#### 2. API | Configure API Key

`.env` API

```bash
# .env
touch .env
```

`.env`

```env
# OpenAI API
OPENAI_API_KEY=your-openai-api-key-here

# OpenRouter OpenAI API
OPENAI_API_KEY=your-openrouter-api-key-here
OPENAI_API_BASE=https://openrouter.ai/api/v1
```

#### 3. | Run Code

```bash
#
source venv/bin/activate

#
python codes/Chapter-01-Prompt-Chaining-Example.py

#
# python codes/Chapter-XX-Example.py
```
