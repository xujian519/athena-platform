# PDF转Markdown工具

## 概述

这是一个专门针对中文PDF文档的优雅Markdown转换工具，特别适合处理政府办事指南、技术文档等结构化PDF文档。

- 专利和集成电路布图设计缴费服务指南.pdf → .md (54KB)
- 专利审查评议平台办事指南.pdf → .md (4KB)
- 专利行政复议财产保全因公调查办事指南.pdf → .md (12KB)

## 功能特点

### 1. 智能页眉页脚清洗
- 自动识别并删除页码（如：- 1 -、— 1 —、第1页、Page 1 of 10等）
- 清除政府文档常见页眉（如：国家知识产权局...）
- 过滤重复标题

### 2. 智能标题识别
- 中文序号标题（一、二、三、）→ 二级标题
- 括号序号标题（（一）（二））→ 三级标题
- 数字序号标题（（1）（2））→ 四级标题
- 关键词标题（使用须知、操作说明等）→ 三级标题

### 3. 段落智能合并
- 自动合并PDF分割的段落
- 识别段落边界（句号、引号等）
- 保持标题与正文的清晰分隔

### 4. 特殊字符处理
- 清除 (cid:xxx) 类型的PDF特殊标记
- 清理多余空格和空行
- 统一引号、破折号等符号

### 5. 格式优雅化
- 添加标准Markdown文档头部
- 生成结构化的标题层级
- 保持适当的段落间距

## 使用方法

### 基本用法

```bash
# 处理指定目录下的所有PDF文件
python3 pdf_to_markdown.py /path/to/pdf/directory

# 使用默认目录 (/Users/xujian/Documents/问答库)
python3 pdf_to_markdown.py
```

### 依赖安装

```bash
# 需要安装 pdfplumber
python3 -m pip install pdfplumber
```

## 转换示例

### 输入PDF
```
知识产权政务服务事项 办事指南.pdf
├── 页眉: 国家知识产权局
├── 正文: 一、专利申请...
├── 页脚: - 1 -
└── 页码: 第1页
```

### 输出Markdown
```markdown
# 知识产权政务服务事项 办事指南

> 本文档由PDF自动转换生成

---

## 一、专利申请

### （一）受理条件

1. 在中国内地有经常居所...
```

## 适用场景
