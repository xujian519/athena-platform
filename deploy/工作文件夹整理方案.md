## 🔧 命名规范

### 📁 文件夹命名规范
```
格式: 客户姓名_年月日
示例: 王玉荣_20251217
```

### 📄 文件命名规范
```
专利文件: 专利名称_文件类型_日期.扩展名
财务文件: 客户姓名_缴费通知_日期.扩展名
技术文件: 项目名称_技术分析_日期.扩展名
审查文件: 申请号_审查轮次_日期.扩展名
```
