# PQAI集成到Athena平台

## 📋 项目概述

基于对PQAI（Patent Quality Artificial Intelligence）项目的深度研究，将其专利检索算法和架构设计集成到Athena平台中。

## 🔍 PQAI架构分析

### 核心组件
```
PQAI架构:
├── core/ (核心算法模块)
│   ├── search.py (检索算法)
│   ├── indexer.py (索引构建)
│   ├── vectorizers.py (向量化)
│   ├── representations.py (表示学习)
│   ├── reranking.py (重排序)
│   └── encoders.py (编码器)
├── services/
│   └── vector_search.py (向量搜索服务)
├── models/ (机器学习模型)
└── plugins/ (插件系统)
```

### 关键技术栈
- **深度学习**: TensorFlow 2.9.3, Keras
- **向量搜索**: FAISS, Annoy, USEarch
- **文本处理**: sentence-transformers, transformers
- **后端服务**: FastAPI, uvicorn
- **数据库**: MongoDB, PostgreSQL
- **缓存**: Redis (通过中间件实现)
