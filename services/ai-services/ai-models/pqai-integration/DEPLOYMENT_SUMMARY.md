## 📊 服务部署详情

### 服务配置
- **服务名称**: PQAI专利检索服务
- **服务地址**: http://localhost:8030
- **服务端口**: 8030
- **运行状态**: ✅ 正常运行

### 核心功能
- **专利检索**: 支持语义、关键词、混合检索策略
- **相似专利查找**: 基于向量相似度的专利推荐
- **索引管理**: 动态构建和更新专利索引
- **性能监控**: 实时服务状态和性能指标

### 技术架构
- **深度学习框架**: PyTorch + Transformers
- **语义模型**: sentence-transformers/all-MiniLM-L6-v2 (384维)
- **向量检索**: FAISS索引
- **Web框架**: FastAPI + Uvicorn
- **API协议**: RESTful HTTP

## 📝 使用指南

### 1. 构建专利索引
```bash
curl -X POST http://localhost:8030/index/build \
  -H "Content-Type: application/json" \
  -d '{"patents": [...], "rebuild": true}'
```

### 2. 专利检索
```bash
curl -X POST http://localhost:8030/search \
  -H "Content-Type: application/json" \
  -d '{
    "query": "人工智能专利检索",
    "top_k": 10,
    "search_type": "hybrid",
    "min_similarity": 0.7
  }'
```

### 3. 相似专利查找
```bash
curl http://localhost:8030/search/similar/PATENT001?top_k=5
```
