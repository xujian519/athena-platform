## 📊 质量分析查询

### Neo4j 查询示例

```cypher
-- 1. 总体统计
MATCH (n) RETURN count(n) as 总节点数;
MATCH ()-[r]->() RETURN count(r) as 总关系数;

-- 2. 质量分层统计
MATCH (e:Entity)
RETURN e.quality_layer, count(e) as 数量
ORDER BY 数量 DESC;

-- 3. 实体类型分布
MATCH (e:Entity)
RETURN e.type, count(e) as 数量
ORDER BY 数量 DESC;

-- 4. 高质量实体查询
MATCH (e:Entity {quality_layer: 'high_quality'})
RETURN e.type, e.value, e.confidence
LIMIT 10;

-- 5. 关系网络分析
MATCH (e1:Entity)-[r:RELATED_TO]->(e2:Entity)
RETURN e1.type, r.type, e2.type, count(*) as 数量
ORDER BY 数量 DESC
LIMIT 20;
```
