### 2. 核心技术架构对比

#### 华为侧移式弯折铰链架构

**权利要求1（独立权利要求）**：
```
1. 柔性屏折叠手机的侧移式弯折铰链，其特征在于：
   包括底座，所述底座上设有对称分布的侧移滑动块，
   所述侧移滑动块前后两端设有对称分布的第一连接臂及第二连接臂，
   且所述侧移滑动块一侧设有第一活动架及第二活动架，
   所述侧移滑动块与所述底座滑动连接，
   所述第一连接臂及第二连接臂分别与所述侧移滑动块联动实现轨迹控制，
   且所述侧移滑动块联动第一活动架同步实现侧移弯折转动；

   所述侧移滑动块上设有销孔及轴销，所述第一活动架上设有连接孔，
   所述轴销通过所述销孔连接定位并穿过所述连接孔
   实现第一活动架与所述侧移滑动块的联动配合；

   所述侧移滑动块上设有滑槽，所述底座上设有滑块，
   所述滑槽与滑块的滑动配合实现侧移滑动块与所述底座的相对滑动。
```

**技术架构解析**：
- **核心组件**：底座 + 侧移滑动块 + 第一/第二连接臂 + 第一/第二活动架
- **运动机制**：侧移滑动（滑槽-滑块配合）+ 轨迹控制（连接臂联动）
- **创新点**：侧移式弯折，通过滑动实现水滴形态

**从属权利要求2-8的关键技术特征**：

| 权利要求 | 技术特征 | 技术效果 |
|---------|---------|---------|
| 权2 | 滑槽为横向或斜滑槽，滑块对应 | 实现平移式或斜移式滑动 |
| 权3 | 条形孔+滑动柱配合 | 侧移滑动块与连接臂联动 |
| 权4 | 齿轮+齿纹啮合 | 0°-180°同步转动 |
| 权5 | 圆弧槽+圆弧块滑动 | 第一/第二活动架联动 |
| 权6 | 凹轮连接板+凸轮配合 | 0°及180°卡点自锁 |
| 权7 | 直滑槽+直滑块 | 水滴形态弯折轨迹控制 |
| 权8 | 条形槽+曲线槽双轨迹 | U型→水滴型过渡 |

**华为核心创新点**：
1. **侧移滑动机制**：通过滑槽-滑块实现侧移滑动
2. **双轨迹控制**：U型拟态 + 水滴型拟态过渡
3. **多方案灵活性**：提供5种连接臂联动方案（方案1-5）

#### 小米三级连杆铰链架构

**权利要求1（独立权利要求）**：
```
1. 一种铰链，其特征在于，包括：
   基体，所述基体沿第一方向延伸；

   分别与所述基体两侧转动连接的两个旋转单元，
   每个所述旋转单元均包括沿第二方向依次排布且转动连接的至少两个转体，
   所述第一方向与所述第二方向垂直；

   分别与所述基体转动连接的两个同步体，
   每个所述同步体均与一个所述旋转单元对应且用于与对应的所述旋转单元同步转动；

   其中，相邻的所述转体间的转动、所述基体与所述旋转单元间的转动
   以及所述同步体与所述基体间的转动所环绕的轴线均沿所述第一方向设置。
```

**技术架构解析**：
- **核心组件**：基体 + 旋转单元（至少2个转体）+ 同步体 + 连接体
- **运动机制**：三级连杆转动 + 滑动补偿 + 齿轮同步
- **创新点**：三级杆组，多自由度，完全闭合

**从属权利要求2-8的关键技术特征**：

| 权利要求 | 技术特征 | 技术效果 |
|---------|---------|---------|
| 权2 | 同步体与转体滑动连接 | 转动时滑动补偿距离变化 |
| 权3 | 连接体两端转动连接 | 牵拉同步体，适应距离变化 |
| 权4 | 第一/第二/第三转体 + 第一转动轴 | 形成2个自由度转动 |
| 权5 | 安装槽 + 第一弧线滑动 | 补偿转动过程长度变化 |
| 权6 | 第一滑槽 + 滑块 | 限定旋转单元位置稳定性 |
| 权7 | 第二滑槽 + 第二转动轴 | 第三转体与同步体滑动 |
| 权8 | 第一齿轮 + 第二齿轮啮合 | 保证两侧同步转动 |

**小米核心创新点**：
1. **三级杆组结构**：第一转体+第二转体+第三转体
2. **滑动补偿机制**：同步体与转体滑动连接，补偿距离变化
3. **多自由度**：至少4个自由度（两侧各2）
4. **可拆卸设计**：拆装方便，便于维护

### 3. 机械结构详细对比

#### 华为侧移式结构

**组件层级**：
```
底座（固定基础）
├── 侧移滑动块（可滑动）
│   ├── 滑槽（横向/斜向）
│   ├── 销孔 + 轴销
│   └── 滑动柱
├── 第一连接臂（轨迹控制）
│   ├── 条形孔/弧形槽
│   └── 齿纹
├── 第二连接臂（轨迹控制）
│   ├── 条形孔/弧形槽
│   └── 凸轮
├── 第一活动架（侧移转动）
│   ├── 连接孔
│   └── 圆弧槽
├── 第二活动架（侧移转动）
│   └── 圆弧块
├── 齿轮（同步转动）
├── 端面连接板
└── 连接轴 + 弹簧（卡点自锁）
```

**运动传递链**：
```
侧移滑动块滑动
    ↓
第一/第二连接臂轨迹控制
    ↓
第一/第二活动架侧移转动
    ↓
齿轮同步转动
    ↓
实现0°-180°弯折（U型→水滴型过渡）
```

**关键设计参数**：
- 弯折范围：0°-180°
- 侧移方式：平移式或斜移式
- 轨迹形态：传统水滴型 或 U型→水滴型过渡
- 同步方式：齿轮啮合
- 自锁机制：凸轮+弹簧卡点自锁

#### 小米三级连杆结构

**组件层级**：
```
基体（沿第一方向延伸）
├── 安装槽（对称，弧形）
├── 第一齿轮（相互啮合，对称）
└── 第三转动轴（对称）
    └── 第二齿轮

旋转单元（两侧对称）
├── 第一转体
│   ├── 第一滑槽（弧形）
│   ├── 连接耳 + 销轴孔
│   └── 连接耳207
├── 第二转体
│   ├── 连接耳 + 销轴孔
│   └── 第二转动轴
└── 第三转体
    ├── 第二滑槽（沿第二方向）
    └── 第二转动轴

同步体（两侧对称）
├── 第二齿轮
├── 第三转动轴
└── 第二滑槽内滑动

连接体（连杆）
├── 一端：与第一/第二转体转动连接
└── 另一端：与同步体转动连接
```

**运动传递链**：
```
旋转单元转动
    ↓
连接体牵拉同步体
    ↓
同步体同步转动 + 滑动补偿
    ↓
第一齿轮 + 第二齿轮啮合
    ↓
实现两侧同步转动 + 完全闭合
```

**关键设计参数**：
- 自由度：至少4个（两侧各2）
- 转体数量：每个旋转单元至少2个（实施例为3个）
- 弯折形态：水滴状截面
- 同步方式：齿轮啮合 + 连接体牵拉
- 可维护性：可拆卸，销轴连接

### 4. 技术实现路径对比

#### 华为的技术路径

**核心技术路线**：
1. **侧移滑动** → 2. **轨迹控制** → 3. **侧移转动** → 4. **同步弯折**

**实现步骤**：
```
步骤1：侧移滑动块在底座上滑动（滑槽-滑块配合）
步骤2：第一/第二连接臂与侧移滑动块联动（轨迹控制）
步骤3：第一/第二活动架同步实现侧移弯折转动
步骤4：齿轮啮合实现0°-180°同步转动
步骤5：凸轮+弹簧实现0°及180°卡点自锁
```

**技术特点**：
- 侧向滑动路径独特
- 双轨迹控制灵活（U型+水滴型）
- 5种连接臂联动方案可选
- 强调轨迹控制的精确性

#### 小米的技术路径

**核心技术路线**：
1. **旋转单元转动** → 2. **连接体牵拉** → 3. **同步体同步** → 4. **滑动补偿** → 5. **齿轮同步**

**实现步骤**：
```
步骤1：旋转单元（第一/第二/第三转体）转动
步骤2：连接体牵拉同步体随旋转单元转动
步骤3：同步体与转体滑动连接，补偿距离变化
步骤4：第一齿轮+第二齿轮啮合，保证两侧同步
步骤5：实现完全闭合和完全展开（水滴状截面）
```

**技术特点**：
- 三级连杆结构清晰
- 滑动补偿机制创新
- 多自由度转动灵活
- 强调可维护性和可拆卸

### 5. 弯折形态与运动轨迹对比

#### 华为的弯折形态

**两种形态模式**：

**模式1：传统水滴形态**
- 实现方式：直滑槽 + 直滑块
- 轨迹特点：全程水滴型拟态
- 适用场景：标准折叠需求

**模式2：U型→水滴型过渡**
- 实现方式：条形槽 + 曲线槽双轨迹
- 轨迹特点：
  - 前段行程：U型拟态
  - 后段行程：水滴型拟态
  - 过渡平滑：U型逐步过渡到水滴型
- 技术优势：结合两者优点

**形态控制示意图**（根据专利描述推断）：
```
展开状态 (180°)
    ↓
[U型阶段] 前段行程，大半径弯折
    ↓
[过渡阶段] U型向水滴型平滑过渡
    ↓
[水滴阶段] 后段行程，小半径弯折
    ↓
闭合状态 (0°)，水滴状截面
```

#### 小米的弯折形态

**单形态模式**：水滴状截面

**实现方式**：
- 三级转体协调转动
- 同步体滑动补偿
- 两侧完全同步

**形态特点**：
```
展开状态 (180°)
    ↓
[三级转动] 第一/第二/第三转体依次转动
    ↓
[滑动补偿] 同步体与第三转体滑动补偿距离
    ↓
[齿轮同步] 两侧完全同步转动
    ↓
闭合状态 (0°)，完全闭合，水滴状截面
```

**关键技术**：
- 闭合后呈现水滴状截面
- 完全闭合，无楔形空隙
- 完全展开，形成平面

### 6. 同步机制对比

#### 华为的同步机制

**同步方案1：齿轮啮合同步**
- 组件：齿轮 + 齿纹
- 位置：第一连接臂之间
- 功能：实现0°-180°弯折范围的同步转动
- 特点：传统齿轮同步

**同步方案2：凹凸配合自锁**
- 组件：凹轮连接板 + 凸轮
- 位置：连接轴 + 第二连接臂
- 功能：实现0°及180°位置的卡点自锁
- 特点：卡点自锁，弹力及扭力控制

**华为同步特点**：
- 依赖齿轮啮合
- 强调位置自锁
- 0°-180°全程同步

#### 小米的同步机制

**同步方案1：齿轮啮合同步**
- 组件：第一齿轮（基体）+ 第二齿轮（同步体）
- 位置：基体第一方向一端
- 功能：两个同步体间接啮合，保证同步转动
- 特点：360°完整齿轮或部分齿轮

**同步方案2：连接体牵拉同步**
- 组件：连接体（连杆）
- 连接方式：
  - 一端：与第一/第二转体转动连接
  - 另一端：与同步体转动连接
- 功能：牵拉同步体随旋转单元同步转动
- 特点：双重同步机制

**小米同步特点**：
- 齿轮啮合 + 连接体牵拉双重同步
- 从首尾两端+中部三节点限制转动
- 同步性和稳定性更高

### 8. 创新点与技术优势对比

#### 华为的创新点

**核心创新**：
1. **侧移式弯折**：独特的侧向滑动路径
2. **双轨迹控制**：U型→水滴型平滑过渡
3. **多方案灵活性**：5种连接臂联动方案
4. **轨迹控制精确性**：精确控制弯折轨迹

- 0°-180°全角度同步转动
- 双轨迹模式，适用不同场景

#### 小米的创新点

**核心创新**：
