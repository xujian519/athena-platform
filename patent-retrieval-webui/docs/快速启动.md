# 专利混合检索系统 - 快速启动指南

## 🎯 5分钟快速上手

### 步骤1: 进入项目目录

```bash
cd patent-retrieval-webui
```

### 步骤2: 一键启动

```bash
./start.sh
```

等待几秒钟，看到以下提示表示启动成功：

```
✅ 前端服务已启动
✅ 后端服务已启动
🎉 系统启动完成！

📍 访问地址：
   前端界面: http://localhost:3000
   后端API:  http://localhost:8000
   API文档:  http://localhost:8000/docs
```

### 步骤3: 打开浏览器

访问 http://localhost:3000

### 步骤4: 开始检索

在搜索框输入查询内容，例如：
```
深度学习图像识别
```

点击"开始检索"，即可看到检索结果！

## 🔍 功能介绍

### 1. 检索

输入关键词，系统会并行执行三种检索策略，智能融合结果。

### 2. 历史

查看所有搜索记录，点击即可重新执行搜索。

### 3. 配置

调整三种检索策略的权重，优化检索效果。

### 4. 监控

实时查看系统性能指标和查询趋势。

## 💡 使用技巧

### 优化检索结果

1. **使用具体关键词**
   - ❌ "好的东西"
   - ✅ "深度学习卷积神经网络"

2. **合理使用筛选条件**
   - IPC分类: G06F（电子数据处理）
   - 申请人: 具体公司名称
   - 日期范围: 缩小搜索时间范围

3. **调整检索权重**
   - 专利侵权分析: 提高向量权重
   - 技术文献检索: 提高全文搜索权重
   - 技术趋势分析: 提高知识图谱权重

## ⚙️ 高级配置

### 修改端口

**前端端口** (默认3000):
编辑 `vite.config.ts`:
```typescript
server: {
  port: 3001, // 修改为其他端口
}
```

**后端端口** (默认8000):
编辑 `backend/api_server.py`:
```python
uvicorn.run(app, port=8001)  # 修改为其他端口
```

### 修改主题颜色

编辑 `tailwind.config.js`:
```javascript
theme: {
  extend: {
    colors: {
      primary: {
        500: '#your-color', // 修改主色调
      }
    }
  }
}
```

## 🐛 常见问题

### Q: 启动失败？

**A:** 检查以下几点：
1. Node.js版本是否 >= 14
2. Python3是否已安装
3. 端口3000和8000是否被占用

### Q: 检索返回空结果？

**A:** 检查：
1. 数据库是否已初始化
2. 查询关键词是否合理
3. 筛选条件是否过于严格

### Q: 前端无法连接后端？

**A:** 检查：
1. 后端服务是否正常运行
2. 浏览器控制台是否有错误信息
3. 防火墙是否阻止了8000端口
