## 📊 技术栈

### 前端
- Vue 3.4+ (Composition API)
- TypeScript 5.3+
- Tailwind CSS 3.4+
- Vue Router 4.2+
- Pinia 2.1+
- ECharts 5.5+
- Axios 1.6+
- Vite 5.0+

### 后端
- FastAPI 0.109+
- Uvicorn 0.27+
- Python 3.8+
- Pydantic 2.5+
- WebSocket 12.0+

### 数据库
- PostgreSQL 12+
- Qdrant 1.0+
- Neo4j 4.0+

```

├── src/                      # 前端源码

│   ├── router/               # 路由配置
│   ├── stores/               # 状态管理（2个）

│   └── main.ts               # 入口文件

│   └── start_server.sh       # 启动脚本
├── docs/                     # 项目文档（7个）
├── index.html                # HTML模板
├── package.json              # 项目配置
├── vite.config.ts            # Vite配置

├── start.sh                  # 一键启动脚本

```
