## 🚀 快速启动步骤

### 1. 安装依赖

```bash
# 前端依赖
cd patent-retrieval-webui
npm install

# 后端依赖会自动安装
```

### 2. 启动服务

```bash
# 方式一：一键启动
./start.sh

# 方式二：分别启动
# 后端
cd backend
./start_server.sh

# 前端（新终端）
npm run dev
```

### 3. 访问系统

- 前端界面: http://localhost:3000
- 后端API: http://localhost:8000
- API文档: http://localhost:8000/docs

## 🎨 界面特色

### 设计亮点

1. **渐变背景**: slate-900 → purple-900 → slate-900
2. **玻璃拟态**: backdrop-blur + 半透明边框
3. **流畅动画**: fadeIn、slideUp、pulse效果
4. **响应式布局**: 支持移动端到桌面端
5. **深色主题**: 护眼舒适的视觉体验
