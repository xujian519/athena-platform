# 专利混合检索系统 - 开发指南

## 🛠️ 开发环境准备

### 系统要求

- Node.js: >= 14.0.0
- Python: >= 3.8
- npm: >= 6.0.0
- Git: >= 2.0

### 安装依赖

```bash
# 安装前端依赖
npm install

# 安装后端依赖
cd backend
pip install -r requirements.txt
```

## 📂 项目结构

```
patent-retrieval-webui/
├── src/                    # 前端源码
│   ├── api/                # API客户端
│   ├── components/         # Vue组件
│   ├── router/             # 路由配置
│   ├── stores/             # 状态管理
│   ├── types/              # TypeScript类型

│   └── main.ts             # 入口文件
├── backend/                # 后端源码

├── docs/                   # 项目文档
├── public/                 # 静态资源
├── index.html              # HTML模板
├── package.json            # 项目配置

```

## 🎯 开发工作流

### 1. 启动开发服务器

```bash
# 启动前端
npm run dev

# 启动后端（新终端）
cd backend
python api_server.py
```

### 2. 开发新功能

#### 添加新组件

```bash
# 1. 创建组件文件
touch src/components/NewComponent.vue

# 2. 在视图中使用
# 在 views/YourView.vue 中：
import NewComponent from '@/components/NewComponent.vue';
```

#### 添加新页面

```bash
# 1. 创建视图文件
touch src/views/NewView.vue

# 2. 添加路由
# 在 router/index.ts 中：
{
  path: '/new',
  name: 'new',
  component: () => import('@/views/NewView.vue')
}

# 3. 添加导航链接
# 在 components/NavBar.vue 中：
<NavLink to="/new">新页面</NavLink>
```

#### 添加新API

```bash
# 1. 在 src/api/client.ts 添加方法
async function newApiMethod() {
  const response = await this.client.get('/new-endpoint');
  return response.data;
}

# 2. 在 src/types/index.ts 定义类型
export interface NewApiResponse {
  // 类型定义
}

# 3. 在Store中调用
// 在 stores/yourStore.ts 中：
const result = await apiClient.newApiMethod();
```

### 3. 代码规范

#### Vue组件风格

```vue
<template>
  <div class="component">
    <!-- 模板内容 -->
  </div>
</template>

<script setup lang="ts">
// 1. 导入依赖
import { ref, computed, onMounted } from 'vue';

// 2. 定义Props
interface Props {
  prop1: string;
  prop2?: number;
}

const props = withDefaults(defineProps<Props>(), {
  prop2: 0
});

// 3. 定义Emits
const emit = defineEmits<{
  change: [value: string];
  submit: [data: any];
}>();

// 4. 响应式状态
const state = ref('');

// 5. 计算属性
const computedValue = computed(() => {
  return state.value.toUpperCase();
});

// 6. 方法
function handleClick() {
  emit('change', state.value);
}

// 7. 生命周期
onMounted(() => {
  // 初始化逻辑
});
</script>

<style scoped>
/* 组件样式 */
</style>
```

#### TypeScript类型定义

```typescript
// 在 src/types/index.ts 中定义公共类型

export interface ExampleData {
  id: string;

  createdAt: Date;
}

export interface ApiResponse<T> {
  data: T;
  message: string;
  status: number;
}

export type ExampleType = 'type1' | 'type2' | 'type3';
```

#### API请求

```typescript
// 在 src/api/client.ts 中添加新API方法

async function customMethod(params: any) {
  try {
    const response = await this.client.post('/custom', params);
    return response.data;
  } catch (error) {
    console.error('API Error:', error);
    throw error;
  }
}
```

## 📝 提交代码

### Git提交规范

```bash
# 格式: type(scope): subject

# type类型:
# feat: 新功能
# fix: 修复bug
# docs: 文档更新
# style: 代码格式
# refactor: 重构
# test: 测试相关
# chore: 构建/工具
```
