### 可复用写作模式

#### 模式1：故事化开场
```
公式: 引用名言 → 描写生活场景 → 提出终极追问

示例:
> 每一个不曾起舞的日子，都是对生命的辜负。
> ——尼采

下班的时候已经很晚，公交车回家。上车三站后，
忽然发现方向反了...

代理人一天究竟在忙啥？我是谁，我究竟为谁而奋斗？
```

#### 模式2：对比式结构
```
公式: 普遍认知（别人）→ 个人实践（自己）→ 反差呈现
```

#### 模式3：数据化证据
```
公式: 时间记录 + 任务清单 + 完成状态
```

#### 模式4：追问式升华
```
公式: 层层递进 + 重复发问 + 价值升华 + 哲学引用收尾

示例结尾:
我是谁，我为谁而奋斗？

[职业身份的自我认知]
[作为行业链条的价值定位]
[知识产权经济时代的使命感]

路正长。。"

最后，以尼采名言作结：
一个人知道自己为什么而活，就可以忍受任何一种生活。
```

#### 模式5：法理引入式开头
```
公式: 引用法律条文 → 阐述基本原理 → 引出实务问题

示例:
专利法第70条规定："为生产经营目的使用、许诺销售或者销售不知道是
未经专利权人许可而制造并售出的专利侵权产品，能证明该产品合法来源的，
不承担赔偿责任。"

根据上述规定，关于销售者是否知情应当根据实际情况综合判断...
```
