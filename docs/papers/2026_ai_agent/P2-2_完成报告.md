### 4. 新增数据结构

```python
# 子任务
@dataclass
class SubTask:
    subtask_id: str
    task_type: PatentTaskType
    description: str
    dependencies: List[str]
    estimated_time: float
    required_tools: List[str]
    priority: ExecutionPriority

# 工作流步骤
@dataclass
class WorkflowStep:
    step_id: str
    stage: WorkflowStage
    subtasks: List[SubTask]
    description: str
    order: int

# 分类结果
@dataclass
class TaskClassificationResult:
    classification_id: str
    primary_task_type: PatentTaskType
    secondary_task_types: List[PatentTaskType]
    complexity: TaskComplexity
    detected_intent: str
    intent_confidence: float
    entities: Dict[str, str]
    subtasks: List[SubTask]
    workflow_steps: List[WorkflowStep]
    recommended_tools: List[str]
    estimated_total_time: float
```

## 使用示例

```python
from core.patent.ai_services.task_classifier import (
    PatentTaskClassifier,
    classify_patent_task,
    format_classification_report,
    PatentTaskType,
    TaskComplexity
)

# 方式1: 便捷函数
result = await classify_patent_task(
    query="请帮我分析这个产品是否侵犯了专利CN12345678A"
)

# 查看结果
print(f"任务类型: {result.primary_task_type.value}")
# 输出: infringement_analysis
print(f"复杂度: {result.complexity.value}")
# 输出: complex
print(f"提取实体: {result.entities}")
# 输出: {'patent_number': 'CN12345678A'}

# 格式化报告
print(format_classification_report(result))
```
