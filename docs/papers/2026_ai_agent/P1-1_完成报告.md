### 2. 核心算法

#### 信息论方法

```
Scope = 1 / Self-Information(claim)
Self-Information = -log₂(P(claim))

关键发现:
- LLM估算概率比字符/词频统计更准确
- 字符数比词数更可靠
```

#### 多因素调整

```python
adjusted_score = raw_score * char_factor * narrowing_factor * broadening_factor

其中:
- char_factor: 基于字符数的调整
- narrowing_factor: 基于缩小因素数量
- broadening_factor: 基于扩大因素数量
```

### 3. 新增类和数据结构

```python
# 范围得分
class ScopeScore:
    raw_score: float          # 原始得分
    normalized_score: float   # 归一化得分 (0-100)
    confidence: float         # 置信度

# 概率估算
class ProbabilityEstimate:
    probability: float        # LLM估算的概率
    self_information: float   # 自信息
    method: str               # 估算方法
    model_used: str           # 使用的模型

# 风险等级
class RiskLevel(Enum):
    LOW = "low"               # 低风险 (>70分)
    MEDIUM = "medium"         # 中风险 (50-70分)
    HIGH = "high"             # 高风险 (30-50分)
    VERY_HIGH = "very_high"   # 极高风险 (<30分)

# 分析结果
class ScopeAnalysisResult:
    claim_text: str
    scope_score: ScopeScore
    probability_estimate: ProbabilityEstimate
    risk_level: RiskLevel
    narrowing_factors: List[str]
    broadening_factors: List[str]
    recommendations: List[str]
```

## 使用示例

```python
from core.patent.ai_services.claim_scope_analyzer import (
    ClaimScopeAnalyzer,
    format_scope_report
)

# 创建分析器
analyzer = ClaimScopeAnalyzer(llm_manager=llm_manager)

# 分析单个权利要求
result = await analyzer.analyze_scope(
    claim_text="1. 一种光伏充电系统，包括光伏板、充电控制器和储能电池...",
    claim_number=1,
    mode="balanced"
)

# 查看结果
print(f"范围得分: {result.scope_score.normalized_score:.1f}/100")
print(f"风险等级: {result.risk_level.value}")
print(f"优化建议: {result.recommendations}")

# 格式化报告
print(format_scope_report(result))
```
