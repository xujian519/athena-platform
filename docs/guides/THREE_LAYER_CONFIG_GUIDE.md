## 🆘 遇到问题？

### 问题1: 配置没有生效

**原因**: 可能是配置路径不对

**解决**:
```bash
# 检查配置文件是否存在
ls -la ~/.claude/CLAUDE.md
ls -la .claude/CLAUDE.md

# 重新加载配置
# 重启Claude Code或重新打开会话
```

### 问题2: 回答不符合预期

**原因**: 可能是配置优先级理解有偏差

**解决**: 在会话开始时明确说明：
```markdown
## 本次会话
请**忽略**项目配置，按照以下要求进行：
{具体要求}
```

### 问题3: 跨项目使用混乱

**解决**: 明确指定项目：
```markdown
## 当前项目
- 项目名称: {实际项目名}
- 技术栈: {实际技术栈}
- 任务目标: {实际目标}

请**忽略**Athena项目配置。
```
