```python
    model = "qwen-max"  # ¥2/百万tokens

# 隐私敏感 → 本地模型
elif is_sensitive:
    model = "local-qwen"
```

### 2. 实现智能缓存

```python
# 缓存常见查询
cache = {}

async def analyze_with_cache(patent_id):
    if patent_id in cache:
        return cache[patent_id]
    
    result = await cloud_llm.generate(f"分析{patent_id}")
    cache[patent_id] = result
    return result
```

**节省**: 50-80%的API调用

### 3. 批量处理优化

```python
# 并发处理
tasks = [
    analyze_patent(pid) 
    for pid in patent_ids
]
results = await asyncio.gather(*tasks)
```

**节省**: 时间成本，提高效率
