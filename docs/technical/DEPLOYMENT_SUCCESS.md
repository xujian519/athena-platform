## 🚀 使用方法

### 1. 小诺平台使用

启动小诺后选择选项6进入智能决策：

```bash
python3 xiaonuo_main.py

# 选择 6 进入智能决策
```

决策类型：
- 技术决策 - 技术选型、架构设计、方案选择
- 产品决策 - 功能优先级、产品策略、版本发布
- 运营决策 - 资源分配、任务安排、团队管理
- 战略决策 - 市场策略、发展规划、投资决策
- 快速决策 - 简单二选、快速判断
- 决策统计 - 查看历史决策和统计
- 决策模板 - 查看可用决策模板

### 2. API接口使用

启动API服务：
```bash
cd /services/api
python3 decision_api.py
```

访问API文档：http://localhost:8000/docs

主要端点：
- POST `/decision` - 执行决策
- POST `/decision/quick` - 快速决策
- GET `/decision/stats` - 决策统计
- GET `/decision/templates` - 决策模板

### 3. Claude Code环境使用

在Claude Code中可以直接使用自然语言命令：

```bash
# 基础决策
帮我在React和Vue之间做选择，考虑团队技能

# 技术决策
需要在PostgreSQL和MongoDB之间选择数据库，支持复杂查询

# 业务决策
这些功能应该优先开发哪个：用户认证、支付系统、数据分析

# 快速决策
应该今天发布还是明天发布？
```
