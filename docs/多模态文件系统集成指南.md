### 2. 平台集成代码

#### Python客户端

```python
import asyncio
from simple_platform_test import AthenaPlatformMultimodal

async def upload_patent_document():
    client = AthenaPlatformMultimodal()

    # 连接
    if await client.connect():
        # 上传专利文档
        result = await client.upload_document(
            "patent_document.pdf",
            doc_type="patent"
        )

        if result.get("success"):
            file_id = result["file_info"]["file_id"]
            print(f"专利文档上传成功，ID: {file_id}")

            # 获取文档信息
            info = await client.get_document_info(file_id)
            print(f"文档信息: {info}")

    await client.close()

# 运行
asyncio.run(upload_patent_document())
```

#### 直接API调用

```python
import requests

# 1. 认证获取token
auth_response = requests.post("http://localhost:8001/auth/login", data={
    "username": "athena_platform",
    "password": "athena_integration_2024"
})
token = auth_response.json()["token"]

# 2. 上传文件
with open("document.pdf", "rb") as f:
    files = {"file": f}
    data = {"token": token, "description": "专利文档"}
    upload_response = requests.post(
        "http://localhost:8001/upload",
        files=files,
        data=data
    )
    file_id = upload_response.json()["file_info"]["file_id"]

# 3. 获取文件信息
info_response = requests.get(
    f"http://localhost:8001/files/{file_id}",
    headers={"Authorization": f"Bearer {token}"}
)
```

### 3. Athena平台集成示例

```python
# 在Athena平台服务中使用
from services.athena_platform.multimodal_integration import platform_multimodal

async def process_patent_filing(patent_data, document_file):
    # 初始化集成
    await platform_multimodal.initialize()

    # 上传专利文档
    upload_result = await platform_multimodal.upload_document(
        document_file,
        doc_type="patent",
        metadata={
            "patent_id": patent_data["id"],
            "title": patent_data["title"],
            "filing_date": patent_data["date"]
        }
    )

    if upload_result["success"]:
        file_id = upload_result["file_info"]["file_id"]

        # 可以进行OCR处理（后续功能）
        ocr_result = await platform_multimodal.process_document(
            file_id,
            process_type="ocr"
        )

        return {
            "file_id": file_id,
            "status": "uploaded",
            "ocr_task": ocr_result.get("task_id")
        }
    else:
        return {"error": upload_result["error"]}
```

## API接口

### 核心端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/auth/login` | POST | 用户认证 |
| `/upload` | POST | 文件上传 |
| `/files` | GET | 文件列表 |
| `/files/{file_id}` | GET | 文件信息 |
| `/files/{file_id}/download` | GET | 下载文件 |
| `/statistics` | GET | 统计信息 |
| `/health` | GET | 健康检查 |

### 集成端点

| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/multimodal/status` | GET | 集成服务状态 |
| `/api/multimodal/upload` | POST | 平台文件上传 |
| `/api/multimodal/files` | GET | 平台文件列表 |
| `/api/multimodal/files/{file_id}` | GET | 平台文件信息 |
| `/api/multimodal/files/{file_id}/download` | GET | 平台文件下载 |

## 故障排除

### 常见问题

1. **服务无法启动**
   - 检查端口8001是否被占用
   - 确认依赖包已安装

2. **认证失败**
   - 检查用户名密码是否正确
   - 确认服务正在运行

3. **文件上传失败**
   - 检查文件大小是否超限
   - 确认存储路径权限
