## 二、详细检索执行步骤

### 2.1 第一阶段：全面检索（预计3-5个工作日）

#### 2.1.1 手动农业工具领域

**Step 1：确定IPC/CPC分类号**
```
主要分类号：
A01B 1/00 手动耕作工具
A01B 3/00 手动松土工具
A01D 1/00 手动收割工具
A01D 7/00 手动割草机
A01D 11/00 手动采摘工具
A01C 7/00 手动播种机
A01C 15/00 手动施肥设备
A01G 3/00 修枝工具（手动）

扩展检索：
B25D 1/00 手持式工具
B25F 5/00 多功能组合工具
```

**Step 2：构建检索式**
```
基础检索式示例：
(IPC=(A01B1/* OR A01B3/* OR A01D1/*)) AND
(发明名称=(手动 OR 手持 OR 便携式) AND
(摘要=(多功能 OR 组合式 OR 可调节) OR
(权利要求=(创新 OR 改进 OR 新型)))

时间限定：2014-2024
法律状态：授权专利
专利类型：实用新型 + 发明
```

**Step 3：检索执行和结果筛选**
- 执行检索，获取初步结果
- 去重和筛选（保留高度相关专利）
- 建立检索结果数据库
- 记录检索历史和结果统计

### 2.2 第二阶段：深度分析（预计5-7个工作日）

#### 2.2.1 新颖性分析

**分析框架：**
```
对于每个待分析专利方案：

1. 技术特征提取
   - 结构特征
   - 功能特征
   - 效果特征

2. 对比文献筛选
   - 选取3-5篇最接近的现有技术
   - 制作对比分析表

3. 差异化分析
   - 明确区别技术特征
   - 评估区别的技术效果
   - 判断是否具备实质性特点
```

**分析报告模板：**
```markdown
## [专利名称] - 新颖性分析报告

### 一、技术方案概述
- 技术领域：
- 解决的技术问题：
- 技术方案核心：
- 预期技术效果：

### 二、检索结果统计
- 检索数据库：
- 检索式：
- 命中文献数：
- 高度相关文献：X篇
- 一般相关文献：Y篇

### 三、对比文献分析
**对比文献1：**
- 专利号：CNXXXXXXXXX.X
- 公开日：YYYY-MM-DD
- 技术要点：
- 相似度：XX%
- 主要区别：

### 四、新颖性结论
- 结论：具备/不具备新颖性
- 理由：
- 建议：
```

#### 2.2.2 创造性分析

**"三步法"分析：**
```
1. 确定最接近的现有技术
2. 确定发明的区别特征和实际解决的技术问题
3. 判断要求保护的发明对本领域技术人员来说是否显而易见
```

**创造性评判要素：**
- 技术突破性
- 技术组合的非显而易见性
- 技术效果的预料不到性
- 商业成功（如有）

#### 2.2.3 实用性分析

**分析要点：**
- 技术方案的可实施性
- 制造工艺的可行性
- 使用场景的适用性
- 经济效益的合理性

## 三、质量控制流程

### 3.1 检索过程质量控制

**检查清单：**
- [ ] 检索策略是否全面
- [ ] 检索数据库是否覆盖充分
- [ ] 时间范围是否合适
- [ ] 关键词是否完整
- [ ] 分类号是否准确
- [ ] 检索结果是否去重
- [ ] 相关性判断是否准确

### 3.2 分析结果质量控制

**双人复核机制：**
1. 第一人完成初稿分析
2. 第二人独立复核
3. 差异点讨论确定
4. 最终报告审定

### 3.3 报告输出标准

**内容要求：**
- 数据准确无误
- 分析逻辑清晰
- 结论明确具体
- 建议切实可行

**格式要求：**
- 统一的报告模板
- 清晰的结构层次
- 必要的图表支持
- 完整的参考文献

### 附件1：检索式示例

```sql
-- 手动农业工具检索式示例
SELECT * FROM patent_db
WHERE (ipc LIKE 'A01B1/%' OR ipc LIKE 'A01D1/%')
  AND (application_date BETWEEN '2014-01-01' AND '2024-12-31')
  AND (patent_type IN ('实用新型', '发明专利'))
  AND (title LIKE '%手动%' OR title LIKE '%手持%' OR title LIKE '%便携式%')
ORDER BY application_date DESC;
```
