# 顶级专家版小娜专利系统架构文档

## 🌟 系统概述

顶级专家版小娜专利系统是一个集成了星河系列专家团队、大语言模型、知识库和专利分析工作流的智能专利分析平台。系统采用多层次架构设计，提供从技术描述到专利分析报告的端到端服务。

**系统版本**: v1.0
**架构模式**: 微服务 + 事件驱动
**技术栈**: Python 3.8+, asyncio, pydantic

### 2. 智能提示词生成系统 (Prompt Generation System)

#### 2.1 专家提示词生成器 (ExpertPromptGenerator)
- **文件**: `core/cognition/expert_prompt_generator.py`
- **功能**: 基于专家类型和分析需求动态生成专业提示词
- **特色**:
  - 支持多种分析类型（专利申请、审查、诉讼、技术评估）
  - 个性化专家指导原则
  - 评估标准和期望输出定义

#### 2.2 提示词模板系统
```python
prompt_templates = {
    'patent_application': "专利申请分析模板",
    'patent_examination': "专利审查分析模板",
    'patent_litigation': "专利诉讼分析模板",
    'technical_evaluation': "技术评估模板"
}
```

### 3. 知识库连接器 (Knowledge Connector)

#### 3.1 知识库架构
- **模拟数据库**: 包含专利、法律先例、技术洞察
- **检索模式**:
  - 相似性搜索 (Similarity Search)
  - 关系搜索 (Relationship Search)
  - 混合搜索 (Hybrid Search)
- **知识类型**:
  - 相似专利数据库
  - 法律先例数据库
  - 技术洞察数据库
  - 领域知识库

#### 3.2 检索算法
- **关键词提取**: 基于NLP的关键词识别
- **相似度计算**: Jaccard相似度算法
- **结果排序**: 相关性分数排序
- **缓存机制**: 查询结果缓存优化

### 4. LLM调用接口 (LLM Interface)

#### 4.1 接口设计
- **文件**: `core/cognition/llm_interface.py`
- **支持的模型**: GPT-4, Claude, 本地模型
- **特色功能**:
  - 专家上下文增强
  - 知识库信息注入
  - 响应质量评估
  - 推理链提取

#### 4.2 请求响应模型
```python
@dataclass
class LLMRequest:
    prompt: str
    model_name: str
    temperature: float
    expert_context: Optional[Dict[str, Any]]
    knowledge_context: Optional[Dict[str, Any]]

@dataclass
class LLMResponse:
    content: str
    confidence_score: float
    reasoning_chain: Optional[List[str]]
    expert_analysis: Optional[Dict[str, Any]]
```

### 5. 专利分析工作流 (Patent Analysis Workflow)

#### 5.1 工作流设计
- **文件**: `core/cognition/xiaona_patent_workflow.py`
- **处理步骤**:
  1. 知识库检索
  2. 专家团队分析
  3. LLM增强分析
  4. 专利性分析
  5. 风险评估
  6. 建议生成
  7. 文档准备

#### 5.2 处理流程图
```
专利分析请求
    │
    ▼
┌─────────────┐
│ 知识库检索   │ ← 查询相似专利、法律先例、技术洞察
└─────────────┘
    │
    ▼
┌─────────────┐
│ 专家团队分析 │ ← 星河专家协同分析
└─────────────┘
    │
    ▼
┌─────────────┐
│ LLM增强分析  │ ← 专家上下文+知识库+LLM
└─────────────┘
    │
    ▼
┌─────────────┐
│ 专利性分析   │ ← 新颖性、创造性、实用性
└─────────────┘
    │
    ▼
┌─────────────┐
│ 风险评估     │ ← 技术、法律、市场风险
└─────────────┘
    │
    ▼
┌─────────────┐
│ 建议生成     │ ← 具体可操作的建议
└─────────────┘
    │
    ▼
┌─────────────┐
│ 报告输出     │ ← 多格式专业报告
└─────────────┘
```

## 📊 数据模型

### 核心数据结构

#### 1. 专利分析请求 (PatentAnalysisRequest)
```python
@dataclass
class PatentAnalysisRequest:
    request_id: str
    invention_title: str

    invention_description: str
    analysis_type: str
    user_requirements: List[str]

    target_jurisdiction: str
```

#### 2. 专利分析结果 (PatentAnalysisResult)
```python
@dataclass
class PatentAnalysisResult:
    request_id: str
    analysis_summary: Dict[str, Any]

    supporting_documents: List[Dict[str, Any]]
```
