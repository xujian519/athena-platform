## 🔧 技术实现方案

### 方案1：统一嵌入服务（推荐）
创建一个全局的BGE嵌入服务，所有模块统一调用：

```python
# core/embedding/unified_embedding_service.py
class UnifiedEmbeddingService:
    """统一嵌入服务"""

    def __init__(self):
        self.bge_service = None  # 延迟初始化
        self.module_configs = {}  # 各模块配置

    async def get_embedding(self, text, module_name="default"):
        """获取嵌入向量"""
        # 根据模块优化参数
        config = self.module_configs.get(module_name, {})
        return await self.bge_service.encode(text, **config)
```

### 方案2：模块化集成
各模块独立集成BGE，保持灵活性。

### 方案3：混合策略
关键模块使用BGE，简单场景使用轻量模型。
