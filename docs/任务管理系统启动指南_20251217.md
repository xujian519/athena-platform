# Athena工作平台 - 任务管理系统启动指南

## 🚀 快速启动

### 1. 启动交互模式
```bash
python3 dev/scripts/start_task_management_system.py
```
**用途**: 日常使用，添加、查看、管理任务

### 2. 启动守护进程
```bash
bash dev/scripts/start_task_daemon.sh
```
**用途**: 后台持续运行，自动提醒和检查任务

### 3. 查看系统状态
```bash
python3 dev/scripts/start_task_management_system.py --stats
```

### 4. 测试提醒功能
```bash
python3 dev/scripts/start_task_management_system.py --test
```

```bash
# 检查进程状态
ps aux | grep start_task_management

# 重启守护进程
bash dev/scripts/stop_task_daemon.sh
bash dev/scripts/start_task_daemon.sh
```

```bash

```

### 检查即将到期任务
```python3
from core.tasks.enhanced_todo_write import get_enhanced_todo
todo = get_enhanced_todo()
due_tasks = todo.get_due_todos(hours_ahead=24)
for task in due_tasks:
    print(f\"⏰ {task['content']} - {task['deadline']}\")
```

## 📝 任务管理API

### 添加新任务
```python
from core.tasks.enhanced_todo_write import add_todo

task_id = add_todo(
    content="新任务内容",
    status="pending",
    priority="high",
    deadline="2025-12-20 15:00",
    reminder_minutes=60,
    tags=["工作", "重要"]
)
```

### 完成任务
```python
from core.tasks.enhanced_todo_write import complete_todo
complete_todo("任务内容")
```

## 🔧 配置管理

### 查看配置
```python
import json
with open('config/task_management.json', 'r') as f:
    config = json.load(f)
    print(f\"提醒间隔: {config['reminders']['default_remind_before']} 分钟\")
```

### 修改配置
```python
# 修改默认提醒时间（分钟）
config['reminders']['default_remind_before'] = 120
with open('config/task_management.json', 'w') as f:
    json.dump(config, f, indent=2)
```
