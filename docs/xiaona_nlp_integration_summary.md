## 💡 使用建议

### 日常使用

1. **启动小娜**:
   ```bash
   # 使用NLP增强版
   bash /Users/xujian/Athena工作平台/production/scripts/start_xiaona_nlp.sh

   # 或直接运行
   cd /Users/xujian/Athena工作平台/production/services
   python3 xiaona_agent_v2_enhanced.py
   ```

2. **API调用**:
   ```python
   from xiaona_agent_v2_enhanced import XiaonaAgentV2Enhanced, QueryRequest

   # 初始化
   agent = XiaonaAgentV2Enhanced(enable_nlp=True)

   # 查询
   request = QueryRequest(
       message="专利法第22条第3款是什么？",
       scenario="general",
       use_rag=True,
       use_nlp=True
   )

   response = agent.query(request)
   print(f"响应: {response.response}")
   print(f"使用的模型: {response.model}")
   print(f"NLP增强: {response.nlp_used}")
   ```

### 配置优化

根据实际使用情况，可调整以下参数：

1. **NLP服务地址**:
   ```python
   nlp_integration = get_xiaona_nlp_integration(
       nlp_service_url="http://localhost:8001"  # 默认
   )
   ```

2. **复杂度阈值**:
   - 当前: 3个复杂度关键词触发complex
   - 建议: 根据实际数据调整

3. **缓存设置**:
   ```python
   # NLP结果缓存
   XiaonaNLPIntegration(enable_cache=True)

   # 提示词缓存
   XiaonaPromptLoaderV2(use_cache=True)
   ```

## 🔧 已知问题和优化方向

### 问题1: 复杂度评估准确性

**现象**: 部分简单查询被识别为复杂查询

**原因**: 关键词匹配策略需要优化

**解决方案**:
1. 引入机器学习模型进行复杂度分类
2. 增加查询长度、结构等多维度评估
3. 基于历史数据动态调整阈值

### 问题2: 意图识别准确率

**现象**: 部分查询意图识别为unknown

**原因**: NLP服务训练数据可能未覆盖专利法律领域

**解决方案**:
1. 使用小娜的真实查询数据增强NLP训练集
2. 添加专利领域专用意图分类器
3. 实现意图识别的本地规则增强

### 问题3: 实体提取准确性

**当前状态**: 使用NLP服务的通用NER

**优化方向**: 实现专利领域专用实体提取
- 专利号: CN/U.S./EP/WO等多种格式
- 法条编号: 支持多种引用格式
- 决定书编号: 识别复审/无效决定书
