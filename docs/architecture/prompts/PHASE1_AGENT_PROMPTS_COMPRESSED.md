# Phase 1 智能体提示词设计（优化版 v5.0）

> **版本**: 5.0 Compressed
> **创建日期**: 2026-04-21
> **优化策略**: 极致压缩 + 模板化 + 参数化

---

## 📋 优化说明

本文档提供Phase 1智能体的**压缩版提示词**，采用以下优化策略：

1. **极致压缩**: 初始加载从20K tokens降至5K tokens
2. **模板化**: 可复用的模板定义
3. **参数化**: 运行时填充参数
4. **渐进式加载**: 核心 → 场景 → 完整
5. **继承v4.0优点**: whenToUse、并行调用、Scratchpad

---

## 🎯 优化效果

| 智能体 | 原设计 | 核心加载 | 完整加载 | 压缩率 |
|--------|--------|---------|---------|--------|
| 检索者 | 20K tokens | **5K tokens** | **8K tokens** | **60%** |
