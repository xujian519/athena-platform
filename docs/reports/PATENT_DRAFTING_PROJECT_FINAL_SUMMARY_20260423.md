## 🎯 使用指南

### 快速开始

```python
# 方式1: 直接导入
from core.agents.xiaona.patent_drafting_proxy import PatentDraftingProxy

agent = PatentDraftingProxy()
result = await agent.execute(context)

# 方式2: 通过xiaona模块
from core.agents.xiaona import PatentDraftingProxy

agent = PatentDraftingProxy()

# 方式3: 通过小诺统一调配
from core.intelligent_collaboration.xiaonuo_coordinator import XiaonuoCoordinator

coordinator = XiaonuoCoordinator()
result = await coordinator.delegate_task(
    task_type="patent_drafting",
    input_data={"disclosure": {...}}
)
```

### 核心能力调用

```python
# 分析技术交底书
result = await agent.analyze_disclosure(disclosure_data)

# 评估可专利性
result = await agent.assess_patentability(disclosure_data)

# 撰写说明书
result = await agent.draft_specification(disclosure_data)

# 撰写权利要求书
result = await agent.draft_claims(disclosure_data)

# 完整撰写流程
result = await agent.draft_patent_application(disclosure_data)
```
