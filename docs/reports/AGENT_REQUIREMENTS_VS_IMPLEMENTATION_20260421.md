## 🎯 待实现智能体详细需求

### 1. 侵权分析智能体（InfringementAnalyzerAgent）

**角色定位**: 专利侵权分析专家

**核心能力**:
1. **权利要求解释**
   - 读取并理解权利要求
   - 确定保护范围
   - 识别必要技术特征

2. **特征比对**
   - 将被诉产品/方法与权利要求比对
   - 确定哪些特征被覆盖
   - 识别等同特征

3. **侵权判定**
   - 全面原则侵权判定
   - 等同原则侵权判定
   - 间接侵权判定

4. **风险评估**
   - 侵权概率评估
   - 赔偿额度估算
   - 规避设计建议

**输入**:
- 目标专利权利要求
- 被诉产品/方法描述
- 技术对比资料

**输出**:
```json
{
  "infringement_analysis": {
    "patent_id": "CN123456789A",
    "product": "被诉产品名称",
    "claims_analysis": [
      {
        "claim_number": 1,
        "infringement": "literal_infringement",
        "covered_features": ["特征1", "特征2"],
        "missing_features": [],
        "equivalent_features": []
      }
    ],
    "infringement_conclusion": "构成侵权",
    "risk_level": "high",
    "suggested_design_around": ["规避方案1", "规避方案2"]
  }
}
```

### 2. 申请文件审查智能体（ApplicationDocumentReviewerAgent）

**角色定位**: 专利申请文件质量审查专家

**核心能力**:
1. **格式规范审查**
   - 检查申请文件格式
   - 核对必备文件清单
   - 验证著录项目信息

2. **技术披露充分性审查**
   - 检查技术问题描述
   - 评估技术方案完整性
   - 审查有益效果描述

3. **权利要求书审查**
   - 检查权利要求格式
   - 评估保护范围合理性
   - 识别明显缺陷

4. **说明书审查**
   - 检查说明书充分性
   - 验证实施例完整性
   - 审查附图规范性

**输入**:

- 相关审查指南

### 3. 撰写审查智能体（WritingReviewerAgent）

**角色定位**: 专利撰写质量审查专家

**核心能力**:
1. **法律用语规范性**
   - 检查法律用语准确性
   - 识别不恰当表述
   - 建议专业用语

2. **技术描述准确性**
   - 审查技术术语使用
   - 验证技术逻辑正确性
   - 检查数据一致性

3. **撰写风格一致性**
   - 检查全文风格统一
   - 评估可读性
   - 确保逻辑连贯

4. **撰写质量评估**
   - 整体质量评分
   - 对比标准案例
   - 提供改进建议

**输入**:
- 专利申请文件草稿
- 撰写规范指南
- 优秀案例参考

**角色定位**: 专利无效宣告分析专家

   - 确定检索关键词
