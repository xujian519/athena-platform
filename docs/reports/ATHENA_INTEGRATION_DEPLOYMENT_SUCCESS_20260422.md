## 回滚方案

### 7.1 如果需要回滚

**短期回滚** (1周内):
```bash
# 回滚到上一个版本
git revert HEAD

# 或回滚到指定提交
git reset --hard 981fbece

# 强制推送
git push github main --force
```

**恢复备份**:
```bash
# 恢复原始备份
cp core/agent/athena_agent.py.backup core/agent/athena_agent.py

# 重新提交
git add core/agent/athena_agent.py
git commit -m "回滚: 恢复原始版本"
git push github main
```
