## 🎯 实施的安全特性

### 1. 进程隔离 ✅

**实现方式**: 使用`asyncio.create_subprocess_exec`在独立进程中执行代码

**效果**:
- 代码在独立的Python进程中运行
- 进程崩溃不影响主程序
- 可以强制终止进程

**代码位置**: `core/tools/code_executor_sandbox.py` 第266-295行

---

### 2. 资源限制 ✅

**实现方式**: 使用`resource.setrlimit`设置资源限制

**限制项**:
- **CPU时间**: 5秒（可配置）
- **内存**: 100MB（可配置）
- **文件大小**: 10MB（可配置）

**代码位置**: `core/tools/code_executor_sandbox.py` 第191-226行

**注意**: 在某些系统上可能不生效（如macOS）

---

### 3. 超时控制 ✅

**实现方式**: 使用`asyncio.wait_for`设置执行超时

**效果**:
- 默认超时：10秒
- 超时自动终止进程
- 防止无限循环

**测试结果**:

### 4. 危险操作阻止 ✅

**实现方式**: 在执行前检查代码中的危险关键词

**阻止的操作**:

- `import subprocess` - 进程执行
- `eval()` - 动态代码执行
- `exec()` - 动态代码执行
- `open()` - 文件操作
- `globals()`, `locals()`, `vars()` - 变量访问
- `getattr()`, `setattr()`, `delattr()` - 对象操作

**测试结果**:


**代码位置**: `core/tools/code_executor_sandbox.py` 第232-257行

### 5. 临时文件隔离 ✅

**实现方式**: 使用`tempfile.TemporaryDirectory`创建临时工作目录

**效果**:

- 执行完成后自动清理
- 环境变量隔离

### 6. 受限的执行环境 ✅

**实现方式**: 重写`__builtins__`，只提供安全函数

**提供的函数**:

```python
__builtins__ = {
```

## 💡 使用方法

### 方式1: 通过Handler（推荐）

```python
from core.tools.code_executor_sandbox_wrapper import code_executor_sandbox_handler

result = await code_executor_sandbox_handler(
    params={
        "code": "print('Hello, World!')",
        "timeout": 10.0,
        "max_memory": 100 * 1024 * 1024
    },
    context={}
)

print(result['output'])  # Hello, World!
```

### 方式2: 通过便捷函数

```python
from core.tools.code_executor_sandbox_wrapper import execute_code_sandbox

result = await execute_code_sandbox(
    code="for i in range(5): print(i)",
    timeout=5.0
)

print(result['output'])
```

### 方式3: 直接使用沙箱

```python
from core.tools.code_executor_sandbox import execute_in_sandbox

result = await execute_in_sandbox(
    code="print('Hello!')",
    timeout=10.0,
    max_memory=50 * 1024 * 1024
)
```
