## 💻 使用示例

### 基础使用

```python
from core.patent.drafting import PatentDrafter, DraftingOptions

# 创建撰写器
drafter = PatentDrafter()

# 配置选项
options = DraftingOptions(
    claim_count=3,  # 生成3条权利要求
    include_background=True,
    include_detailed_description=True,
    writing_style="formal"
)

# 撰写专利申请
application = await drafter.draft_patent_application(
    disclosure_file="技术交底书.txt",
    options=options
)

# 输出结果
print(f"发明名称: {application.title}")
print(f"摘要: {application.abstract}")
print(f"权利要求: {len(application.claims)}条")

# 保存到文件
application.save_to_file("专利申请.txt")
```

### 批量撰写

```python
# 批量处理多个技术交底书
disclosure_files = [
    "交底书1.txt",
    "交底书2.txt",
    "交底书3.txt"
]

applications = await drafter.batch_draft(
    disclosure_files,
    options=options
)

print(f"成功撰写 {len(applications)} 件专利申请")
```
