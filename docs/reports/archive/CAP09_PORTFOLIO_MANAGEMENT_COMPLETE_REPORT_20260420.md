- ✅ 市场潜力评估（0-1）

- ✅ 决策理由生成

### 新建云熙Agent v2.0

**文件**: `core/agents/yunxi_ip_agent.py`
**基于**: BaseAgent v2.0统一接口
**职责**: IP管理和专利组合管理

**核心能力**:
1. **专利组合管理** - 清单维护、分类分级
2. **专利价值评估** - 动态评估、维持建议
3. **期限跟踪** - 年费提醒、到期预警

**Agent能力**:
```python
capabilities = [
    "portfolio-management",  # 专利组合管理
    "patent-valuation",      # 专利价值评估
    "deadline-tracking",      # 期限跟踪
    "classification_grading", # 分类分级
    "maintenance_decision"    # 维持决策
]
```

**使用示例**:
```python
agent = YunxiIPAgent()
await agent.initialize()

# 分析专利
request = AgentRequest(
    request_id="test_001",
    action="portfolio-management",
    parameters={"patent_id": "CN123456789A", "action": "analyze"}
)

response = await agent.process(request)
```
