## 一、迁移策略

### 1.1 分阶段迁移

考虑到系统中有1344个潜在工具，采用**分阶段、分优先级**的迁移策略：

**阶段1: 核心工具迁移** (P0 - 1周)
- 11个核心工具
- 涉及专利检索、文档处理、网络搜索等关键功能

**阶段2: 常用工具迁移** (P1 - 2-3周)
- 317个分析工具
- 274个搜索工具
- 按使用频率和重要性排序

**阶段3: 其他工具迁移** (P2 - 按需)
- 742个其他工具
- 根据实际需求逐步迁移

### 1.2 迁移原则

1. **向后兼容**: 保留旧API，提供shim层
2. **增量迁移**: 一次迁移一个工具，测试验证后再继续
3. **优先级驱动**: 优先迁移使用频率高的工具
4. **自动化辅助**: 使用脚本辅助，但保留人工审核

### 2.2 迁移步骤

对于每个工具，执行以下步骤：

**步骤1**: 添加@tool装饰器
```python
from core.tools.decorators import tool

@tool(
    name="patent_search",
    category="patent_search",
    description="在专利数据库中搜索专利",
    priority="high",
    tags=["search", "patent", "database"]
)
def patent_search_handler(query: str, limit: int = 10) -> list[dict]:
    """在专利数据库中搜索专利"""
    # 原实现
    pass
```

**步骤2**: 更新导入路径
```python
# 旧代码
from core.tools.base import ToolRegistry
registry = ToolRegistry()
registry.register_tool("patent_search", handler)

# 新代码
from core.tools.unified_registry import get_unified_registry
registry = get_unified_registry()
# 使用@tool装饰器自动注册
```

**步骤3**: 测试验证
```python
# 验证工具已注册
from core.tools.unified_registry import get_unified_registry
registry = get_unified_registry()
tool = registry.get("patent_search")
assert tool is not None

# 验证工具功能
result = tool.function(query="人工智能", limit=5)
assert len(result) <= 5
```
