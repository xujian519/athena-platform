## 工具功能概述

### 核心能力

1. **风险识别** - 基于场景和风险因子自动识别风险等级
2. **概率评估** - 估算风险发生的概率范围
3. **影响分析** - 评估风险发生后的影响程度
4. **风险评分** - 为每个风险计算量化评分
5. **缓解策略** - 生成针对性的风险缓解建议
6. **整体评估** - 汇总所有风险，给出整体风险等级

### 支持的风险类型

工具内置了5种常见风险类型：

| 风险类型 | 默认描述 | 典型缓解策略 |
|---------|---------|-------------|
| **技术风险** | 技术实现困难或失败 | 加强技术调研,进行原型验证,准备备用方案 |
| **时间风险** | 项目延期或时间不足 | 制定详细计划,设置里程碑,预留缓冲时间 |
| **资源风险** | 人力、资金或设备不足 | 提前规划资源分配,建立资源储备,优化资源配置 |
| **市场风险** | 市场需求变化或竞争加剧 | 持续市场调研,灵活调整策略,建立竞争优势 |
| **政策风险** | 政策法规变化影响 | 关注政策动向,合规经营,建立应对预案 |

## 算法评估

### 风险评分算法

```python
def _risk_score(risk_level: str) -> int:
    """风险评分"""
    scores = {"高": 8, "中": 5, "低": 2}
    return scores.get(risk_level, 5)
```

**评分规则**:
- 高风险: 8分
- 中风险: 5分
- 低风险: 2分

### 整体风险等级计算

```python
# 计算平均风险评分
avg_score = total_score / len(risk_factors)

# 根据平均评分确定整体风险等级
if avg_score >= 7:
    overall_risk_level = "高"
    risk_color = "red"
elif avg_score >= 4:
    overall_risk_level = "中"
    risk_color = "yellow"
else:
    overall_risk_level = "低"
    risk_color = "green"
```

**阈值设计**:
- 高风险: 平均分 ≥ 7
- 中风险: 4 ≤ 平均分 < 7
- 低风险: 平均分 < 4
