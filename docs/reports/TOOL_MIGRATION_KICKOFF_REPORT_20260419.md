## 三、技术架构

### 3.1 迁移工具

**1. @tool装饰器** (`core/tools/decorators.py`)
```python
@tool(
    name="tool_name",
    category="category",
    description="工具描述",
    priority="high",
    tags=["tag1", "tag2"]
)
def tool_function(param1: str, param2: int = 10) -> dict:
    """工具函数"""
    pass
```

**2. 统一注册表** (`core/tools/unified_registry.py`)
- 单例模式
- 懒加载机制
- 健康监控
- 统计信息

**3. 向后兼容层** (`core/tools/base.py`)
- 保留旧API
- 自动转发到新注册表
- 平滑迁移

### 3.2 迁移步骤

对于每个工具：

**步骤1**: 添加@tool装饰器
**步骤2**: 更新函数签名
**步骤3**: 更新文档字符串
**步骤4**: 测试验证
**步骤5**: 更新调用方
