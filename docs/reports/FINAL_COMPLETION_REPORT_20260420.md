```go
}
```

**智能分组**:
```go
// 按集合名称和优先级分组

```

**请求合并**:
```go
// 相似度>95%自动合并
similarity = dotProduct(vec1, vec2) / (norm(vec1) * norm(vec2))

    mergedReq.Limit *= len(similarVectors)
}
```
