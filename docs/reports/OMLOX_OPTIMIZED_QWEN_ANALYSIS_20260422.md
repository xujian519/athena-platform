### 三大核心技术

#### 1. **oQ量化**（Optimized Quantization）

**功能**：
- 数据驱动的混合精度权重量化
- 支持3/4/5/6/8-bit量化
- 4x模型压缩，近无损精度

**优势**：
- 自动敏感度测量
- GPTQ优化（Hessian指导的误差补偿）
- 关键层（首层、末层、输出头）使用更高精度

**配置示例**：
```python
quantizer = OQQuantizer("path/to/model")
stats = await quantizer.quantize(
    output_path="path/to/output",
    q_bits=4,  # 生产环境推荐
    q_group_size=64,
)
```

#### 2. **TurboQuant**（KV Cache量化）

**功能**：
- KV Cache极量化（3-bit/4-bit）
- Metal内核加速
- 动态量化模式切换

**内存节省**：
- 4-bit：25%节省
- 3-bit：37.5%节省

**动态策略**：
```
上下文 < 5000:    bf16（未量化）
5000 ≤ 上下文 < 20000:  4-bit
上下文 ≥ 20000:   3-bit
```

#### 3. **DFlash**（块扩散投机解码）

**功能**：
- Draft + Verify循环
- 16个tokens并行生成
- 单次验证

**性能**：
- **加速比**：4-6x（短上下文）
- **接受率**：85-90%
- **速度**：180-200 tok/s

**工作流程**：
```
1. Draft模型并行生成16 tokens
2. Target模型单次验证
3. 接受最长匹配前缀
4. 回退拒绝的tokens
5. 重复直到完成
```
