## 工具集场景映射

### 专利检索场景
```
用户输入: "检索关于自动驾驶的专利"
    │
    ▼
┌───────────────┐
│ 专利检索工具集  │
│ patent_search  │
└───────┬───────┘
        │
        ├─── enhanced_patent_search (CNIPA/USPTO/EPO)
        ├─── google_scholar_search (学术论文)
        ├─── pdf_patent_parser (PDF解析)
        └─── web_search (补充检索)
```

### 新颖性分析场景
```
用户输入: "分析专利CN123456789A的新颖性"
    │
    ▼
┌──────────────────┐
│ 新颖性分析工具集   │
│ novelty_analysis  │
└────────┬─────────┘
         │
         ├─── pdf_patent_parser (解析对比文件)
         ├─── technical_feature_extractor (提取特征)
         ├─── claim_comparator (权利要求对比)
         └─── legal_reasoning_engine (三步法推理)
```

### 审查意见答复场景
```
用户输入: "生成审查意见答复"
    │
    ▼
┌──────────────────┐
│ 审查意见答复工具集 │
│ oa_response       │
└────────┬─────────┘
         │
         ├─── oa_analyzer (分析审查意见)
         ├─── legal_case_search (检索判例)
         ├─── claim_amender (修改权利要求)
         └─── response_generator (生成答复)
```
