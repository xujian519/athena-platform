## 📝 使用说明

### 查看任务状态
```bash
# 查看所有任务
task list

# 查看特定任务
task get <task_id>

# 更新任务状态
task update <task_id> --status in_progress
task update <task_id> --status completed
```

### 开始开发
1. 从Task #1开始（Phase 1: 基础框架）
2. 按依赖关系顺序执行
3. 每完成一个任务，更新状态为completed
4. 遇到阻塞时，记录到任务的metadata中

### 代码提交
- 每完成一个Task，提交一次代码
- 提交格式: `feat(patent-drafting): 实现TechnicalDisclosureAnalyzer`
- 关联任务编号: `Refs #2`
