## 实施详情

### P0: Grafana仪表板（60分钟） ✅

**文件**: `config/grafana/dashboards/tracing-dashboard.json`

**面板清单**:

1. **请求量趋势 (QPS)** - 时间序列图
   - 数据源: Elasticsearch
   - 指标: traceID cardinality
   - 刷新频率: 5秒

2. **延迟分布 (P50/P95/P99)** - 统计面板
   - 数据源: Elasticsearch
   - 阈值: 绿色(<100ms), 黄色(100-500ms), 红色(>500ms)

3. **错误率** - 状态面板
   - 数据源: Elasticsearch
   - 阈值: 绿色(<1%), 黄色(1-5%), 红色(>5%)

4. **Agent平均响应时间** - 条形图
   - 按服务名称分组
   - 显示Top 10服务

5. **Agent请求量分布** - 饼图
   - 按服务名称聚合
   - 显示请求占比

6. **最慢的操作 (Top 10)** - 表格
   - 按span名称分组
   - 显示平均耗时和调用次数

**特性**:
- 服务过滤器模板变量
- 5秒自动刷新
- 直接链接到Jaeger UI
- 深色主题
