1. **任务调度器** (TaskScheduler)

## 🏗️ 系统架构

### 核心组件
```
任务管理系统 (Task Management System)
├── 任务调度器 (TaskScheduler)
│   ├── 任务模型 (Task)
│   ├── 通知管理 (NotificationManager)
│   ├── 存储管理 (TaskStorage)
│   └── 调度循环 (Scheduler Loop)
├── 增强TodoWrite (EnhancedTodoWrite)
│   ├── 持久化存储
│   ├── 兼容性接口
│   ├── 自动保存
│   └── 任务同步
└── 启动脚本 (Start Scripts)
    ├── 交互模式
    ├── 守护进程模式
    └── 测试模式
```

### 数据流
```
任务创建 → 存储到JSON → 调度器同步 → 提醒检查 → 系统通知
     ↓           ↓            ↓           ↓         ↓
   TodoWrite  →  文件系统  →  任务数据库  →  定时循环  →  用户界面
```

```python

@dataclass
class Reminder:
    reminder_type: ReminderType    # 通知类型
```
