## 🏗️ 新架构概览

### 技术栈对比

| 层级 | 旧架构（Neo4j） | 新架构（NebulaGraph） |
|------|----------------|---------------------|
| **图数据库** | Neo4j Community | NebulaGraph |
| **Python驱动** | neo4j-python | nebula3-python |
| **查询语言** | Cypher | nGQL |
| **连接协议** | Bolt | TCP |
| **分布式** | 企业版功能 | 开源支持 |
| **节点ID** | 自动生成 | 手动指定VID |

### 服务架构

```
┌─────────────────────────────────────────────────────────────┐
│                    Athena专利知识图谱系统                    │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  ┌─────────────────────┐         ┌───────────────────────┐ │
│  │   API层（新）       │         │   数据层（新）        │ │
│  │                     │         │                       │ │
│  │ • RESTful API       │         │ • Qdrant向量库       │ │
│  │ • WebSocket支持     │    ←→   │ • NebulaGraph图库    │ │
│  │ • 查询缓存          │         │ • PostgreSQL全文库   │ │
│  └─────────────────────┘         └───────────────────────┘ │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐   │
│  │              混合检索引擎                             │   │
│  │  向量检索(50%) + 全文检索(40%) + 图检索(10%)          │   │
│  └──────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```
