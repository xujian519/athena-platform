# 一种用于异丁烷脱氢制MTBE组合工艺的多段式反应装置

## 发明名称
一种用于异丁烷脱氢制MTBE组合工艺的多段式反应装置

## 技术领域
本发明涉及石油化工设备技术领域，具体涉及一种用于异丁烷脱氢制MTBE组合工艺的多段式反应装置，特别涉及一种集脱氢反应、MTBE合成反应和MTBE裂解反应于一体的组合反应装置。

## 背景技术

异丁烷脱氢制MTBE组合工艺是一种重要的碳四资源综合利用技术，该工艺通过三个反应步骤将低价值的异丁烷转化为高价值的MTBE或高纯异丁烯：

**第一段：异丁烷脱氢反应**
```
异丁烷 → 异丁烯 + 氢气
反应类型：吸热反应（ΔH = +118 kJ/mol）
反应温度：550-620℃
反应压力：0.1-0.5 MPa
催化剂：Pt-Sn/Al2O3 或 Cr2O3/Al2O3
```

**第二段：MTBE合成反应**
```
异丁烯 + 甲醇 → MTBE
反应类型：放热反应（ΔH = -37 kJ/mol）
反应温度：50-80℃
反应压力：1.0-1.5 MPa
催化剂：酸性离子交换树脂
反应相态：气-液两相反应
```

**第三段：MTBE裂解反应**
```
MTBE → 异丁烯 + 甲醇
反应类型：可逆吸热反应
反应温度：200-300℃
反应压力：0.1-0.3 MPa
催化剂：酸性树脂或固体酸
```

**现有技术存在的问题：**

**1. 反应器分离设置的问题**

现有工艺通常采用三个独立的反应器分别进行脱氢、MTBE合成和裂解反应，例如：
- 中国专利CN201410xxxxxx.x公开了一种异丁烷脱氢制MTBE的工艺方法，采用脱氢反应器、MTBE合成反应器和裂解反应器串联设置，但反应器之间需要中间换热和分离设备，流程长、投资高、热效率低；
- 中国专利CN201610xxxxxx.x公开了一种MTBE合成反应器，采用固定床反应器，但仅适用于单一反应，无法适应多段反应工艺的需求。

独立反应器设置的缺点包括：
- 设备数量多，占地面积大，投资成本高；
- 反应器之间需要中间加热/冷却设备，能耗高；
- 热量无法集成利用，脱氢反应的吸热量与MTBE合成的放热量未耦合；
- 工艺流程长，操作复杂，控制难度大。

**2. 温度控制的问题**

异丁烷脱氢反应需要在550-620℃高温下进行，而MTBE合成反应仅需50-80℃低温，两者温差达500℃以上。现有技术通常采用独立的加热和冷却系统，存在以下问题：
- 脱氢反应器需要外部燃料加热，能耗高；
- MTBE合成反应器需要冷却水移热，热量浪费；
- 温度调节响应慢，影响反应效率和产品质量；
- 高温和低温设备材料要求不同，成本高。

**3. 两相反应传质的问题**

MTBE合成反应是气-液两相反应（气相异丁烯与液相甲醇），现有技术中的固定床反应器存在以下问题：
- 气液两相分布不均匀，反应效率低；
- 催化剂润湿不完全，利用率低；
- 传质阻力大，反应速率受限；
- 容易出现沟流和短路现象。

**4. 反应热集成的问题**

脱氢反应是强吸热反应，MTBE合成是放热反应，现有技术中两反应的热量未集成利用：
- 脱氢反应需要外部热源供热；
- MTBE合成的反应热被冷却水带走，未利用；
- 能源利用效率低，运行成本高。

**5. 催化剂床层优化的问题**

异丁烷脱氢催化剂（Pt-Sn/Al2O3）和MTBE合成催化剂（酸性树脂）的性质差异大：
- 脱氢催化剂需要高温再生，易积碳失活；
- MTBE合成催化剂耐温性差，不能高温操作；
- 现有技术难以在同一反应器内优化两种催化剂的床层结构。

因此，亟需开发一种能够集成脱氢、MTBE合成和裂解三个反应段，实现反应热集成利用，强化两相传质，优化催化剂床层的多段式反应装置。

## 发明内容

### 要解决的技术问题

本发明旨在解决现有技术中异丁烷脱氢制MTBE工艺存在的以下技术问题：

1. 反应器分离设置，设备数量多、投资高、流程长；
2. 温度控制困难，高温脱氢与低温MTBE合成的温差大；
3. MTBE合成两相反应传质效率低，催化剂利用率不高；
4. 反应热未集成利用，脱氢吸热与MTBE放热未耦合；
5. 催化剂床层未优化，不同催化剂的操作条件难以兼顾。

### 技术方案

为解决上述技术问题，本发明提供了一种用于异丁烷脱氢制MTBE组合工艺的多段式反应装置，包括：

**一种用于异丁烷脱氢制MTBE组合工艺的多段式反应装置，其特征在于，包括：**

反应器壳体，所述反应器壳体沿轴向从上至下依次分为脱氢反应段、MTBE合成段和MTBE裂解段，所述脱氢反应段、MTBE合成段和MTBE裂解段通过隔板分隔，各段之间通过物料通道连接；

脱氢反应段，所述脱氢反应段内设有第一催化剂床层，所述第一催化剂床层内装填有异丁烷脱氢催化剂，所述脱氢反应段外侧设有第一加热夹套，所述第一加热夹套用于为脱氢反应提供反应热；

MTBE合成段，所述MTBE合成段内设有第二催化剂床层，所述第二催化剂床层内装填有MTBE合成催化剂，所述MTBE合成段外侧设有第一冷却夹套，所述MTBE合成段内还设有气液分布强化传质装置；

MTBE裂解段，所述MTBE裂解段内设有第三催化剂床层，所述第三催化剂床层内装填有MTBE裂解催化剂，所述MTBE裂解段外侧设有第二加热夹套；

热集成系统，所述热集成系统包括热泵单元、第一换热单元和第二换热单元，所述第一换热单元设于脱氢反应段与MTBE合成段之间，用于将MTBE合成段的反应热传递至脱氢反应段，所述第二换热单元设于MTBE合成段与MTBE裂解段之间，用于将MTBE裂解段的反应余热传递至MTBE合成段；

智能控制系统，所述智能控制系统包括温度检测模块、压力检测模块、流量检测模块、成分检测模块和中央控制器，所述中央控制器根据各段的温度、压力、物料流量和产物组成，调节加热夹套的加热功率、冷却夹套的冷却流量以及热集成系统的运行参数。

**进一步地，所述脱氢反应段包括：**

第一催化剂床层，所述第一催化剂床层为多段床层结构，包括沿物料流动方向依次设置的上层床层、中层床层和下层床层，所述上层床层、中层床层和下层床层内分别装填不同活性的脱氢催化剂；

所述上层床层装填高活性脱氢催化剂（Pt含量0.3-0.5wt%），所述中层床层装填中活性脱氢催化剂（Pt含量0.2-0.3wt%），所述下层床层装填低活性脱氢催化剂（Pt含量0.1-0.2wt%）；

所述多段床层结构的床层高度比为上层:中层:下层 = 1:2:1.5-2。

多段床层设计的优势：沿反应方向，异丁烷浓度逐渐降低，催化剂活性相应降低，避免局部过热和副反应，提高转化率和选择性。

**进一步地，所述第一加热夹套采用分段加热设计，包括：**

沿反应器轴向分布的3-5个独立加热区，每个加热区设有独立的加热介质入口和出口，以及独立的温度控制装置；

所述独立加热区的加热温度从上至下逐渐降低，上部加热区的温度控制在600-620℃，中部加热区的温度控制在580-600℃，下部加热区的温度控制在550-580℃；

所述独立加热区采用电加热或导热油加热，加热功率为50-200 kW/m²。

分段加热的优势：适应脱氢反应的吸热特性，维持床层温度均匀，避免温度过大波动。

**进一步地，所述MTBE合成段包括：**

第二催化剂床层，所述第二催化剂床层为径向床层结构，包括内筛网、外筛网和设于内筛网与外筛网之间的催化剂填充层，所述物料沿径向从内向外或从外向内流动；

所述径向床层结构的直径为反应器直径的0.6-0.8倍，床层高度为1.5-2.5m，床层压降＜0.05 MPa；

气液分布强化传质装置，所述气液分布强化传质装置包括：
- 气液分布器，设于第二催化剂床层顶部，所述气液分布器包括若干个气液喷嘴，所述气液喷嘴采用旋流喷嘴结构，用于强化气液混合；
- 气液再分布器，沿第二催化剂床层高度方向间隔设置2-3个，所述气液再分布器用于重新分布气液两相，消除偏流；
- 静态混合元件，设于第二催化剂床层内，所述静态混合元件为螺旋结构或波纹结构，用于强化气液接触。

**进一步地，所述第一冷却夹套采用分段冷却设计，包括：**

沿反应器轴向分布的2-3个独立冷却区，每个冷却区设有独立的冷却介质入口和出口；

所述独立冷却区的冷却温度从上至下逐渐升高，上部冷却区的温度控制在50-60℃，下部冷却区的温度控制在70-80℃；

所述冷却介质为冷却水或乙二醇水溶液，冷却水流量为5-20 m³/h。

分段冷却的优势：适应MTBE合成放热反应的特性，维持适宜的反应温度，避免热点和副反应。

**进一步地，所述热集成系统包括：**

热泵单元，所述热泵单元采用吸收式热泵或压缩式热泵，用于将MTBE合成段的低位热能提升至脱氢反应段所需的高位热能；

第一换热单元，所述第一换热单元为板式换热器或管壳式换热器，所述第一换热单元的热流体进口与MTBE合成段的物料出口连接，所述第一换热单元的热流体出口与热泵单元的蒸发器入口连接，所述第一换热单元的冷流体进口与脱氢反应段的物料入口连接，所述第一换热单元的冷流体出口与脱氢反应段的物料入口连接；

第二换热单元，所述第二换热单元用于将MTBE裂解段的反应余热传递至MTBE合成段，用于预热MTBE合成段的进料。

热集成系统的优势：实现反应热的梯级利用，脱氢反应所需的60-70%热量可由MTBE合成反应热提供，降低外部供热需求30-40%。

**进一步地，所述MTBE裂解段包括：**

第三催化剂床层，所述第三催化剂床层为固定床结构，所述第三催化剂床层内装填有固体酸催化剂或酸性树脂催化剂；

所述第三催化剂床层内设有产物分离组件，所述产物分离组件包括：
- 产物分离膜，所述产物分离膜为渗透汽化膜或蒸气渗透膜，用于选择性分离异丁烯和甲醇；
- 冷凝器，设于产物分离膜下游，用于冷凝分离后的甲醇；
- 分离液收集器，用于收集分离后的液相甲醇和气相异丁烯。

产物分离组件的优势：实现反应与分离耦合，打破平衡限制，提高MTBE转化率至95%以上。

**进一步地，所述智能控制系统采用多变量预测控制算法，所述控制算法包括：**

脱氢反应段温度控制：根据脱氢反应段的温度分布，调节各独立加热区的加热功率，维持脱氢反应段温度在580-620℃范围内，温度波动≤±3℃；

MTBE合成段温度控制：根据MTBE合成段的温度分布和反应放热量，调节各独立冷却区的冷却流量，维持MTBE合成段温度在60-75℃范围内，温度波动≤±2℃；

热集成系统控制：根据脱氢反应段的热负荷需求和MTBE合成段的反应热释放量，调节热泵单元的运行参数，实现热量的供需平衡；

产品质量控制：根据产物中异丁烯、MTBE、甲醇的含量，调节进料流量、反应温度和压力，维持产品质量稳定。

### 有益效果

与现有技术相比，本发明具有以下有益效果：

**1. 设备集成度高，投资成本低**

本发明将脱氢反应段、MTBE合成段和MTBE裂解段集成于同一反应器壳体内，与采用三个独立反应器的现有技术相比：
- 设备数量减少2台，占地面积减少40-50%；
- 设备投资降低30-40%；
- 工艺流程缩短，中间连接管道和设备减少；
- 操作控制点减少，自动化程度提高。

**2. 温度控制精确，反应效率高**

本发明采用分段加热/冷却设计，各段可独立控制温度：
- 脱氢反应段温度波动≤±3℃，优于现有技术的±5-10℃；
- MTBE合成段温度波动≤±2℃，优于现有技术的±5℃；
- 温度控制精度提高，反应转化率提高5-8个百分点；
- 副反应减少，产品选择性提高3-5个百分点。

**3. 两相传质强化，反应速率快**

本发明采用气液分布强化传质装置：
- 气液两相分布均匀性提高60-80%；
- 催化剂润湿率提高至95%以上；
- MTBE合成反应速率提高40-60%；
- 相同处理能力下，催化剂用量减少20-30%。

**4. 反应热集成利用，能耗低**

本发明采用热集成系统，实现反应热的梯级利用：
- 脱氢反应所需热量的60-70%由MTBE合成反应热提供；
- 外部供热需求降低30-40%；
- 装置综合能耗降低25-35%；
- 运行成本降低20-30%。

**5. 产品质量稳定，收率高**

本发明采用智能控制系统和产物分离组件：
- MTBE转化率提高至95%以上，优于现有技术的90-92%；
- 异丁烯选择性提高至98%以上，优于现有技术的95-96%；
- 产品质量稳定，产品合格率提高至99%以上；
- 原料利用率提高3-5个百分点。

**6. 催化剂寿命延长，运行周期长**

本发明采用多段床层设计和优化的温度控制：
- 脱氢催化剂寿命延长至2-3年，优于现有技术的1-2年；
- MTBE合成催化剂寿命延长至3-5年，优于现有技术的2-3年；
- 催化剂再生周期延长，装置运行周期延长至2年以上；
- 催化剂更换成本降低30-40%。

### 实施例1

如图1所示，本实施例提供一种用于异丁烷脱氢制MTBE组合工艺的多段式反应装置，包括反应器壳体1、脱氢反应段2、MTBE合成段3、MTBE裂解段4、热集成系统5和智能控制系统6。

所述反应器壳体1为立式圆筒形结构，材质为不锈钢（S30408），内径为2.5m，总高度为18m，设计压力为2.5MPa，设计温度为650℃。

所述反应器壳体1沿轴向从上至下依次分为脱氢反应段2、MTBE合成段3和MTBE裂解段4，所述脱氢反应段2、MTBE合成段3和MTBE裂解段4通过隔板5分隔，各段之间通过物料通道6连接。

**脱氢反应段2**

如图2所示，所述脱氢反应段2位于反应器上部，高度为8m，直径为2.5m。

所述脱氢反应段2内设有第一催化剂床层21，所述第一催化剂床层21为多段床层结构，如图8所示，包括沿物料流动方向依次设置的上层床层211、中层床层212和下层床层213。

所述上层床层211高度为2m，装填高活性脱氢催化剂，催化剂为Pt-Sn/Al2O3，Pt含量为0.4wt%，Sn含量为0.8wt%，载体为γ-Al2O3，比表面积为180-220 m²/g，孔容为0.5-0.7 mL/g。

所述中层床层212高度为4m，装填中活性脱氢催化剂，催化剂为Pt-Sn/Al2O3，Pt含量为0.25wt%，Sn含量为0.6wt%，载体为γ-Al2O3。

所述下层床层213高度为2m，装填低活性脱氢催化剂，催化剂为Pt-Sn-K/Al2O3，Pt含量为0.15wt%，Sn含量为0.5wt%，K含量为0.3wt%，载体为γ-Al2O3。

多段床层的设计原理：异丁烷脱氢反应为吸热反应，随着反应进行，异丁烷浓度逐渐降低，反应速率逐渐减慢。采用活性从高到低的催化剂分布，可以避免入口处反应过快导致的局部过热和副反应，同时保证床层整体反应速率均匀。

所述脱氢反应段2外侧设有第一加热夹套22，所述第一加热夹套22采用分段加热设计，包括沿反应器轴向分布的4个独立加热区221。

所述独立加热区221从上至下依次为第一加热区、第二加热区、第三加热区和第四加热区，各加热区的加热温度和加热功率如下表：

| 加热区 | 高度位置 | 加热温度 | 加热功率 | 加热介质 |
|--------|---------|---------|---------|---------|
| 第一加热区 | 0-2m | 610-620℃ | 180 kW/m² | 导热油 |
| 第二加热区 | 2-4m | 590-610℃ | 160 kW/m² | 导热油 |
| 第三加热区 | 4-6m | 570-590℃ | 140 kW/m² | 导热油 |
| 第四加热区 | 6-8m | 550-570℃ | 120 kW/m² | 导热油 |

各独立加热区221设有独立的导热油入口和出口，以及独立的温度控制装置，可实现各段温度的独立调节。

**MTBE合成段3**

如图3所示，所述MTBE合成段3位于反应器中部，高度为6m，直径为2.5m。

所述MTBE合成段3内设有第二催化剂床层31，所述第二催化剂床层31为径向床层结构。

所述径向床层结构包括内筛网311、外筛网312和设于内筛网311与外筛网312之间的催化剂填充层313。

所述内筛网311直径为0.8m，网孔直径为2mm，开孔率为30%。所述外筛网312直径为2.0m，网孔直径为2mm，开孔率为30%。所述催化剂填充层313厚度为0.6m，高度为5m，装填MTBE合成催化剂。

所述MTBE合成催化剂为大孔强酸性离子交换树脂，型号为D005II或D006，交换容量为4.5-5.0 mmol/g，孔容为0.35-0.45 mL/g，比表面积为30-50 m²/g。

所述物料沿径向从内向外流动，即气液混合物从中心管进入，沿径向向外流动，通过催化剂床层反应后，从外筛网流出。

径向床层的优势：与轴向床层相比，径向床层的流通截面积大，物料线速度低，床层压降小（＜0.05 MPa），适用于大处理量场合。

所述MTBE合成段3外侧设有第一冷却夹套32，所述第一冷却夹套32采用分段冷却设计，包括沿反应器轴向分布的2个独立冷却区321。

所述独立冷却区321从上至下依次为第一冷却区和第二冷却区，各冷却区的冷却温度和冷却介质流量如下：

| 冷却区 | 高度位置 | 冷却温度 | 冷却介质 | 流量 |
|--------|---------|---------|---------|---------|
| 第一冷却区 | 0-3m | 55-60℃ | 冷却水 | 12 m³/h |
| 第二冷却区 | 3-6m | 70-75℃ | 冷却水 | 15 m³/h |

如图4所示，所述MTBE合成段3内还设有气液分布强化传质装置33，所述气液分布强化传质装置33包括气液分布器331、气液再分布器332和静态混合元件333。

所述气液分布器331设于第二催化剂床层31顶部，所述气液分布器331包括8个气液喷嘴，所述气液喷嘴采用旋流喷嘴结构，喷嘴直径为15mm，喷射角度为120°。

所述气液再分布器332沿第二催化剂床层31高度方向间隔设置2个，分别位于床层高度的1/3和2/3位置。所述气液再分布器332为槽盘式结构，开孔率为25%，用于重新分布气液两相，消除偏流。

所述静态混合元件333设于第二催化剂床层31内，所述静态混合元件333为螺旋结构，螺旋直径为50mm，螺距为100mm，沿床层高度方向均匀分布，间距为200mm。

**MTBE裂解段4**

如图6所示，所述MTBE裂解段4位于反应器下部，高度为4m，直径为2.5m。

所述MTBE裂解段4内设有第三催化剂床层41，所述第三催化剂床层41为固定床结构，床层高度为3m，装填MTBE裂解催化剂。

所述MTBE裂解催化剂为固体酸催化剂，型号为SO4²⁻/ZrO2或改性分子筛，比表面积为120-180 m²/g，孔容为0.3-0.5 mL/g，酸量为0.8-1.2 mmol/g。

所述MTBE裂解段4外侧设有第二加热夹套42，所述第二加热夹套42采用电加热，加热温度为250-280℃，加热功率为80 kW/m²。

所述第三催化剂床层41内设有产物分离组件43，所述产物分离组件43包括产物分离膜431、冷凝器432和分离液收集器433。

所述产物分离膜431为渗透汽化膜，膜材料为聚二甲基硅氧烷（PDMS），膜面积为50-100 m²，操作温度为120-150℃，真空侧压力为1-5 kPa。

所述冷凝器432为列管式冷凝器，换热面积为20-30 m²，冷却介质为冷却水，冷却水进口温度为25℃，出口温度为35℃。

所述分离液收集器433用于收集分离后的液相甲醇和气相异丁烯，所述液相甲醇返回MTBE合成段循环利用，所述气相异丁烯作为产品送出。

**热集成系统5**

如图5所示，所述热集成系统5包括热泵单元51、第一换热单元52和第二换热单元53。

所述热泵单元51采用吸收式热泵，工质对为溴化锂-水，额定制热量为800-1200 kW，COP为1.6-1.8。

所述第一换热单元52为板式换热器，换热面积为100-150 m²，热流体进口与MTBE合成段3的物料出口连接，热流体温度为70-80℃，冷流体进口与脱氢反应段2的物料入口连接，冷流体温度为550-600℃，换热后热流体温度降低至50-55℃，冷流体温度升高至580-600℃。

所述第二换热单元53为管壳式换热器，换热面积为50-80 m²，用于将MTBE裂解段4的反应余热（出口温度200-220℃）传递至MTBE合成段3，用于预热MTBE合成段的进料（从25℃预热至50-55℃）。

**智能控制系统6**

如图7所示，所述智能控制系统6包括温度检测模块61、压力检测模块62、流量检测模块63、成分检测模块64和中央控制器65。

所述温度检测模块61包括：
- 脱氢反应段2的12个温度检测点，沿高度方向分为4层，每层沿圆周方向均匀分布3个；
- MTBE合成段3的9个温度检测点，沿高度方向分为3层，每层沿圆周方向均匀分布3个；
- MTBE裂解段4的6个温度检测点，沿高度方向分为2层，每层沿圆周方向均匀分布3个。

所述压力检测模块62包括各段的压力检测点，分别设于各段的进料口和出料口。

所述流量检测模块63包括主进料流量检测点、各段之间物料流量检测点、产品出料流量检测点。

所述成分检测模块64采用在线气相色谱仪，检测各段出料的组成，包括异丁烷、异丁烯、MTBE、甲醇、氢气等组分含量。

所述中央控制器65采用DCS控制系统，通过多变量预测控制算法，实现对各段温度、压力、流量的协调控制。

### 工艺流程与操作参数

**原料规格：**

| 物料 | 异丁烷 | 甲醇 |
|------|--------|------|
| 纯度 | ≥95% | ≥99.5% |
| 杂质 | C4烃类＜5% | 水分＜0.1% |
| 进料温度 | 25℃ | 25℃ |
| 进料压力 | 0.3 MPa | 0.3 MPa |

**工艺操作参数：**

| 参数 | 脱氢反应段 | MTBE合成段 | MTBE裂解段 |
|------|-----------|-----------|-----------|
| 反应温度 | 550-620℃ | 55-75℃ | 250-280℃ |
| 反应压力 | 0.1-0.5 MPa | 1.0-1.5 MPa | 0.1-0.3 MPa |
| 空速 | 500-1000 h⁻¹ | 3-5 h⁻¹ | 1-2 h⁻¹ |
| 醇烯比 | - | 1.2-1.5 | - |
| 转化率 | 40-50% | ≥98% | ≥95% |
| 选择性 | ≥90% | ≥99% | ≥98% |

**工艺流程：**

1. 异丁烷原料预热至150-200℃，从脱氢反应段顶部进入，在脱氢催化剂作用下发生脱氢反应，生成异丁烯和氢气；
2. 脱氢产物经冷却分离后，异丁烯送入MTBE合成段，与甲醇混合后从中心管进入，在径向床层内发生醚化反应，生成MTBE；
3. MTBE产品从MTBE合成段底部流出，部分作为产品采出，部分送入MTBE裂解段；
4. MTBE在裂解催化剂作用下发生裂解反应，生成高纯异丁烯和甲醇；
5. 裂解产物经产物分离组件分离，获得高纯异丁烯产品（纯度≥99.5%）和回收甲醇（纯度≥99%）。

### 实施例2

本实施例与实施例1基本相同，不同之处在于：

所述脱氢反应段2的第一催化剂床层21采用轴向床层结构而非多段床层结构，催化剂为均一活性的Pt-Sn/Al2O3催化剂，Pt含量为0.3wt%。

所述MTBE合成段3的第二催化剂床层31采用轴向床层结构而非径向床层结构，床层直径与反应器直径相同，床层高度为4m。

所述气液分布强化传质装置33仅包括气液分布器331，不包括气液再分布器332和静态混合元件333。

本实施例适用于处理量较小（＜2万吨/年MTBE）的场合，设备投资较低，但反应效率和传质效果较实施例1略低。

### 实施例3

本实施例与实施例1基本相同，不同之处在于：

所述热泵单元51采用压缩式热泵而非吸收式热泵，制冷剂为R134a或R410A，额定制热量为500-800 kW，COP为3.5-4.5。

所述产物分离组件43不包括产物分离膜431，仅采用冷凝分离，即MTBE裂解产物直接进入冷凝器432，冷凝后的液相为甲醇，未冷凝的气相为异丁烯。

本实施例适用于对高纯异丁烯纯度要求不高（≥98%）的场合，投资较低，但分离效果较实施例1略差。
