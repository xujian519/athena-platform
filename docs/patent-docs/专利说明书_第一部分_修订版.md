# 专利说明书（第一部分：修订版）

---

**发明名称**

一种带有强化传热元件的管壳式换热器

---

**技术领域**

本实用新型涉及换热设备技术领域，具体涉及一种管壳式换热器，尤其涉及一种应用于石油化工领域的、适用于抽余碳五生产异戊烯工艺的带有强化传热元件的管壳式换热器。

---

**背景技术**

管壳式换热器是一种广泛应用于石油化工、煤化工、电力、制冷等行业的热交换设备，其基本结构包括壳体、换热管束、管板、封头以及设置在壳体内的折流板或扰流元件。在工作过程中，一种流体在换热管内流动（管程），另一种流体在壳体内绕流换热管外表面（壳程），通过换热管壁进行热量传递。

在抽余碳五生产异戊烯的工艺中，管壳式换热器是关键设备之一。抽余碳五是碳五分离装置的副产物，其中含有20-30%的异戊烯（包括2-甲基-1-丁烯和2-甲基-2-丁烯），是生产高纯度异戊烯的重要原料来源。目前主流的异戊烯生产工艺采用醚化-裂解法，该工艺主要包括以下步骤：

（1）**加氢预处理**：在温度低于100℃的条件下，对抽余碳五进行选择性加氢，脱除环戊二烯等易聚合杂质，该反应为放热反应，反应热需要及时移出，需使用冷却器控制反应温度在适宜范围；

（2）**醚化反应**：加氢后的抽余碳五与甲醇在温度低于70℃的条件下进行醚化反应，生成甲基叔戊基醚（TAME），该反应也为放热反应，需要精确控制反应温度以保证转化率和选择性；

（3）**TAME分离提纯**：对生成的TAME进行分离提纯，获得高纯度的TAME中间产品；

（4）**高温裂解**：将TAME在220-230℃的高温条件下进行裂解反应，分解为异戊烯和甲醇，该反应为吸热反应，需要输入热量维持反应温度，该过程对温度控制精度要求极高；

（5）**产物分离**：对裂解产物进行分离，获得纯度≥99%的异戊烯产品，分离过程需要多级冷却和冷凝。

上述工艺过程中，抽余碳五及其加工产物具有以下特殊物理化学性质：

**第一，组分沸点接近，分离难度大。** 抽余碳五中异戊烯的各异构体（2-甲基-1-丁烯和2-甲基-2-丁烯）以及正戊烷、异戊烷等组分沸点差异较小（通常在20-40℃之间），这对精馏塔的换热提出了较高要求，需要换热器能够提供精确、均匀的温度控制，确保各分离段的温度稳定在设计范围内，通常要求温控精度达到±1℃。

**第二，介质易聚合、易结垢。** 抽余碳五中含有环戊二烯、双烯烃等易聚合组分，虽然在加氢预处理中大部分被脱除，但残留的双烯烃在高温条件下仍可能发生聚合反应，生成的聚合物附着在换热管表面形成污垢层，显著降低传热效率。此外，在醚化和裂解过程中，如果温度控制不当，也可能导致副反应发生，生成高沸点聚合物。因此，换热器需要具备良好的抗结垢性能，且便于定期清洗维护。

**第三，介质粘度随温度变化显著。** 碳五馏分的粘度对温度较为敏感，在较低温度（如40℃以下）时粘度明显增大，导致流动阻力上升，换热性能下降。因此，在低温换热区域（如产物冷凝器）需要优化流道结构，避免流动死角，减少压降。

**第四，存在腐蚀风险。** 抽余碳五中可能残留微量的硫化物、有机酸等腐蚀性组分，在某些工艺条件下可能对换热器材料产生腐蚀作用。换热器结构应避免死区，防止腐蚀性介质积聚。

为了提高管壳式换热器的传热效率，现有技术中采用了多种强化传热措施。常见的强化传热技术包括：

（1）**改变换热管表面结构**：如采用波纹管、螺旋槽管、翅片管等异形管，通过增加换热面积和扰动流体来提高传热系数。但这类技术往往导致流动阻力显著增加，且制造成本较高。

（2）**改进折流板结构**：如中国专利CN201320508689.X公开的一种管壳式换热器，其换热烟道管上套装有若干层密封扰流板，密封扰流板上开有烟道管安装孔与漏孔，漏孔设于密封扰流板一侧的边缘位置。该结构虽然在一定程度上提高了换热效果，但边缘漏孔的设置较为单一，流体分布不够均匀，且该专利主要针对烟气换热，未考虑碳五馏分易结垢的特点。

（3）**优化折流板开孔**：如中国专利CN201921189625.1（公开号：CN210718357U）公开的一种管壳式换热器和空调机组，其折流板设有第一通孔和第二通孔，第二通孔邻近折流板与壳体的连接处，用于消除滞流区域。该技术虽然能够减少滞流区，但第二通孔为均匀分布的圆孔，导流效果有限，在处理高粘度介质（如低温碳五馏分）时，折流板与壳体之间的角隅区域仍存在流动死区。

（4）**快速拆装结构**：如中国专利CN201821113286.4（公开号：CN208635629U）公开的一种管壳式换热器密封结构，通过卡紧块、螺纹杆和弹簧等组件实现管板与管箱的快速连接，便于设备维护。但该结构主要解决密封连接问题，对传热效率提升作用有限，且未考虑热膨胀补偿问题。

（5）**双出口变工况控制**：如中国专利CN201910818135.1（公开号：CN110345669A）公开的一种管壳式换热器及其控制方法，在壳体上设置壳程第一出口和壳程第二出口，通过调节第二出口的位置来适应变工况运行。该技术虽然提高了变工况适应性，但结构较为复杂，增加了制造成本和控制难度。

然而，将上述现有换热器应用于抽余碳五生产异戊烯工艺时，仍存在以下不足之处：

**第一，传热效率与流动阻力的矛盾难以协调，难以满足精确温控要求。** 抽余碳五生产异戊烯工艺中的换热器（特别是加氢反应器的冷却器和裂解反应器的加热器）需要对介质温度进行精确控制（通常要求温控精度达到±1℃）。现有的强化传热技术往往侧重于单一方面的强化，虽然能够提高传热系数，但同时导致流动阻力大幅增加，使得流体分布不均，反而影响了温度控制的精确性。例如，在醚化反应器中，如果壳程流体分布不均，可能导致局部过热或过冷，影响反应选择性和产物纯度。

**第二，滞留区消除不彻底，易结垢，影响设备长周期运行。** 尽管部分现有技术（如CN201921189625U）通过在折流板边缘设置通孔来消除滞留区，但通孔通常为均匀分布的简单圆孔，导流能力有限，折流板与壳体之间的角隅区域仍存在流动死区。在处理抽余碳五等易结垢介质时，这些死区容易形成物料沉积和聚合物附着，导致传热效率下降，流阻增加，甚至造成堵塞，影响装置的长周期稳定运行。特别是在低温换热区域（如产物冷凝器），介质粘度增大，死区问题更为突出。

**第三，热膨胀补偿不足，影响设备安全运行。** 抽余碳五生产异戊烯工艺中，部分换热器需要在较大的温差范围内运行。例如，加氢反应器的冷却器需要处理80-90℃的介质，而产物冷凝器则需要处理40℃以下的介质，设备在运行过程中会产生显著的热膨胀。现有技术中，换热管束通常采用刚性固定方式，缺乏有效的热膨胀补偿机制。当壳体与换热管束之间存在较大温差时，两者会产生不同步的热膨胀，长期运行容易产生热应力，导致管板连接处泄漏或换热管弯曲变形，影响设备安全运行和使用寿命。

**第四，维护清洗不便，影响装置连续生产。** 由于抽余碳五介质易结垢的特性，换热器需要定期进行清洗维护。然而，现有管壳式换热器的内部强化传热元件通常采用固定安装方式（如焊接或不可拆卸的连接），当换热管内表面或壳程内部结垢时，难以进行有效的清洗维护。特别是在工业化装置中，频繁的停机清洗会严重影响装置的开工率和经济效益。

**第五，结构适应性较差，难以满足不同工艺段的需求。** 抽余碳五生产异戊烯工艺包含多个换热单元，不同单元的工艺条件差异较大。例如，加氢反应器的冷却器需要处理80-90℃、中等粘度的介质，而TAME裂解反应器的加热器则需要处理220-230℃的高温介质，产物冷凝器则需要处理40℃以下、高粘度的介质。现有换热器通常针对特定的工况条件设计，缺乏对不同工艺条件的适应性，难以在整个工艺流程中实现优化配置。

因此，有必要研发一种新型管壳式换热器，针对抽余碳五生产异戊烯工艺的特殊需求，在提升传热效率的同时控制流动阻力，有效消除滞留区，具备热膨胀补偿能力，便于维护清洗，并具有良好的工况适应性，以满足不同工艺段的换热需求，提高装置的运行稳定性和经济效益。
