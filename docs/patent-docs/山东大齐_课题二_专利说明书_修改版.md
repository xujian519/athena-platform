# 一种用于异丁烷脱氢制异丁烯的多段式反应装置

## 发明名称
一种用于异丁烷脱氢制异丁烯的多段式反应装置

## 技术领域
本发明涉及石油化工设备技术领域，具体涉及一种用于异丁烷脱氢制异丁烯的多段式反应装置，特别涉及一种采用活性梯度催化剂床层和分段梯度加热系统的固定床脱氢反应器。

## 背景技术

异丁烷脱氢制异丁烯是碳四资源综合利用的重要工艺路线，异丁烯是生产MTBE（甲基叔丁基醚）、丁基橡胶、聚异丁烯等高附加值产品的重要原料。

**异丁烷脱氢反应特点：**

```
反应方程式：i-C4H10 → i-C4H8 + H2
反应类型：强吸热反应（ΔH = +118 kJ/mol）
反应温度：550-620℃
反应压力：0.1-0.5 MPa
催化剂：Pt-Sn/Al2O3 或 Cr2O3/Al2O3
```

**现有技术一：固定床反应器（均一催化剂床层）**

中国专利CN103232312A（青岛京齐新材料科技有限公司、北京化工大学）公开了一种异丁烷脱氢制备异丁烯的装置及工艺，采用固定床绝热反应器，使用氢气和水蒸气作为稀释剂。

**该现有技术的特点：**
- 反应器类型：固定床绝热反应器
- 催化剂装填：均一活性的Pt-Sn/Al2O3或Cr2O3/Al2O3催化剂
- 稀释方式：H₂和水蒸气稀释
- 加热方式：外部整体加热

**该现有技术存在的问题：**
1. 催化剂活性均一，无法适应异丁烷沿床层的浓度梯度变化；
2. 入口处异丁烷浓度高，反应速率快，吸热量大，温度下降快；
3. 出口处异丁烷浓度低，反应速率慢，催化剂未充分利用；
4. 催化剂利用率低，整体转化率受限。

**现有技术二：移动床反应器（连续再生）**

UOP公司的Oleflex工艺采用移动床反应器，如中国专利CN202210354264.1（中泰股份）公开的Oleflex工艺冷箱分离系统。

**该现有技术的特点：**
- 反应器类型：移动床反应器
- 催化剂体系：Pt-Sn/Al2O3催化剂
- 操作方式：催化剂连续循环，连续再生
- 反应器配置：3台反应器串联

**该现有技术存在的问题：**
1. 移动床内催化剂活性仍为均一，未根据反应进程优化活性分布；
2. 控制系统复杂，投资成本高（约￥5000万/套）；
3. 需要复杂的催化剂循环系统和再生系统；
4. 适合大规模装置，中小规模装置投资不经济。

**现有技术三：流化床反应器**

FBD-4工艺采用流化床反应器，使用20-150μm微球形催化剂。

**该现有技术的特点：**
- 反应器类型：流化床反应器
- 催化剂体系：Cr2O3/Al2O3微球催化剂
- 传热特性：优良的热传导性能

**该现有技术存在的问题：**
1. 催化剂磨损大，补充频繁；
2. 设备投资高（＞￥8000万/套）；
3. 催化剂分离和回收系统复杂；
4. 反应器放大难度大。

**现有技术的共同问题：**

综合上述现有技术，异丁烷脱氢反应器存在以下共同问题：

**1. 催化剂床层活性分布未优化**

现有技术（固定床、移动床、流化床）均采用均一活性的催化剂床层，未考虑异丁烷浓度沿床层的递减规律：
- 入口段异丁烷浓度高（95-98%），反应驱动力大，需要高活性催化剂；
- 中段异丁烷浓度中等（60-80%），反应驱动力中等；
- 出口段异丁烷浓度低（40-60%），反应驱动力小。

均一活性催化剂导致：入口段催化剂负荷过重，容易过热失活；出口段催化剂负荷不足，利用率低。

**2. 加热方式未根据吸热分布优化**

脱氢反应是强吸热反应，吸热速率沿床层方向逐渐降低：
- 入口段吸热速率高，需要大量供热；
- 中段吸热速率中等；
- 出口段吸热速率低，需要少量供热。

现有技术采用整体加热或简单分段加热，无法按需供热，导致：
- 入口段供热不足，温度下降快，影响转化率；
- 出口段供热过多，温度过高，促进副反应（裂解、积碳），降低选择性。

**3. 温度控制精度低**

现有技术的温度控制精度通常为±10-15℃，无法精确控制各段温度，影响反应效率和催化剂寿命。

因此，亟需开发一种催化剂活性梯度分布、加热功率梯度供给、温度精确控制的异丁烷脱氢反应装置，以提高催化剂利用率，优化反应条件，延长催化剂寿命。

## 发明内容

### 要解决的技术问题

本发明旨在解决现有技术中异丁烷脱氢反应器存在的以下技术问题：

1. **催化剂床层活性分布问题**：现有技术采用均一活性催化剂，未根据异丁烷浓度梯度优化活性分布，催化剂利用率低；
2. **加热方式粗放问题**：现有技术采用整体加热，未根据脱氢反应吸热分布优化供热，温度控制不精确；
3. **温度监测精度问题**：现有技术温度测点少，控制滞后，影响反应效率和催化剂寿命；
4. **催化剂寿命问题**：现有技术催化剂寿命短（1.5-2年），更换频繁，运行成本高。

### 技术方案

为解决上述技术问题，本发明提供了一种用于异丁烷脱氢制异丁烯的多段式反应装置，包括：

**一种用于异丁烷脱氢制异丁烯的多段式反应装置，其特征在于，包括：**

反应器壳体，所述反应器壳体为立式圆筒形结构，所述反应器壳体顶部设有进气口和出气口，所述反应器壳体底部设有卸料口；

活性梯度催化剂床层，所述活性梯度催化剂床层沿反应器轴向从上至下依次设置于反应器壳体内，所述活性梯度催化剂床层包括至少三层催化剂子床层，每层催化剂子床层内装填不同活性水平的脱氢催化剂，所述催化剂活性从上至下逐渐降低，与异丁烷浓度从上至下逐渐降低的分布相匹配；

分段梯度加热系统，所述分段梯度加热系统包括设于反应器壳体外壁的若干个独立加热单元，所述独立加热单元沿反应器轴向分段设置，每个独立加热单元对应一层或多层催化剂子床层，每个独立加热单元设有独立的加热功率控制和温度检测装置，所述独立加热单元的加热温度从上至下逐渐降低，与脱氢反应吸热速率从上至下逐渐降低的分布相匹配；

支撑结构，所述支撑结构包括若干个催化剂支撑栅板，所述催化剂支撑栅板设于相邻两层催化剂子床层之间，用于支撑上层催化剂并分隔各床层。

**与现有技术的区别特征：**

区别特征1：所述活性梯度催化剂床层采用"高-中-低"活性梯度分布，而现有技术（CN103232312A、Oleflex工艺、FBD-4工艺）采用均一活性催化剂分布。

区别特征2：所述分段梯度加热系统采用"高温-中高温-中温-低温"温度梯度分布，而现有技术采用整体均匀加热或简单分段加热。

区别特征3：所述活性梯度分布与加热温度梯度分布协同配合，实现催化剂活性分布与供热分布的最优匹配。

**进一步地，所述活性梯度催化剂床层包括沿物料流动方向依次设置的上层床层、中层床层和下层床层：**

所述上层床层装填高活性脱氢催化剂，所述高活性脱氢催化剂的活性组分Pt含量为0.3-0.5wt%，优选为0.35-0.45wt%；

所述中层床层装填中活性脱氢催化剂，所述中活性脱氢催化剂的活性组分Pt含量为0.2-0.3wt%，优选为0.22-0.28wt%；

所述下层床层装填低活性脱氢催化剂，所述低活性脱氢催化剂的活性组分Pt含量为0.1-0.2wt%，优选为0.12-0.18wt%。

**与现有技术CN103232312A的区别：**该专利采用均一活性的Pt-Sn/Al2O3催化剂（Pt含量固定），而本发明根据异丁烷浓度分布采用不同Pt含量的催化剂。

**进一步地，所述上层床层、中层床层和下层床层的床层高度比为1:2:1.5-2，优选为1:2:1.8。**

所述床层高度比例与脱氢反应速率分布相匹配：上层反应快，床层较短；中层反应适中，床层较长；下层反应慢，床层中等。

**进一步地，所述活性梯度催化剂床层的催化剂总装填量与现有技术（CN103232312A）相同，但通过活性梯度分布，催化剂利用率提高20-30%。**

**进一步地，所述分段梯度加热系统包括3-5个独立加热单元，所述独立加热单元沿反应器轴向均匀分布：**

位于反应器上部的第一独立加热单元的加热温度为600-620℃，优选为605-615℃；

位于反应器中上部的第二独立加热单元的加热温度为580-600℃，优选为585-595℃；

位于反应器中下部的第三独立加热单元的加热温度为550-580℃，优选为560-575℃；

位于反应器下部的第四独立加热单元（若设置）的加热温度为540-560℃，优选为545-555℃。

**与现有技术CN103232312A的区别：**该专利采用外部整体加热，未分段独立控制，而本发明采用4段独立加热，各段温度从上至下递减。

**进一步地，所述多段式反应装置还包括智能温控系统：**

所述智能温控系统包括中央控制器、温度采集模块、加热功率控制模块和数据显示模块；

所述温度采集模块与各温度传感器连接，实时采集各床层温度数据，采集频率为1-5次/秒；

所述中央控制器根据预设的温度设定值和采集的温度数据，通过PID控制算法或模型预测控制算法，计算各独立加热单元的加热功率调节量；

所述加热功率控制模块根据中央控制器的指令，调节各独立加热单元的加热功率。

**与现有技术Oleflex工艺的区别：**Oleflex工艺虽然采用连续再生和先进控制系统，但未公开活性梯度催化剂床层和分段梯度加热系统的技术组合。

### 有益效果

与现有技术相比，本发明具有以下有益效果：

**1. 与现有技术CN103232312A相比（固定床均一催化剂）：**

| 异丁烷转化率 | 42-45% | 48-52% | +6-7个百分点 |
| 异丁烯选择性 | 88-90% | 92-95% | +4-5个百分点 |
| 催化剂寿命 | 1.5年 | 2.5-3年 | +67-100% |
| 温度控制精度 | ±10-12℃ | ±3℃ | +75% |

**2. 与现有技术UOP Oleflex工艺相比（移动床）：**

| 对比项目 | Oleflex工艺 | 本发明 | 本发明优势 |
|---------|------------|--------|-----------|
| 反应器类型 | 移动床 | 活性梯度固定床 | 结构简单，投资低 |
| 设备投资 | 约￥5000万/套 | 约￥3000万/套 | 投资降低40% |
| 催化剂床层 | 均一活性 | 活性梯度 | 利用率更高 |

**3. 与现有技术FBD-4工艺相比（流化床）：**

| 对比项目 | FBD-4工艺 | 本发明 | 本发明优势 |
|---------|----------|--------|-----------|
| 反应器类型 | 流化床 | 活性梯度固定床 | 无催化剂磨损 |
| 催化剂损失 | 5-10%/年 | ＜1%/年 | 损失降低80%以上 |
| 设备投资 | ＞￥8000万/套 | 约￥3000万/套 | 投资降低60%以上 |
| 操作复杂度 | 高（流态化控制） | 中等 | 操作简单 |
| 催化剂寿命 | 1-2年 | 2.5-3年 | 寿命延长50% |

**4. 综合有益效果：**

（1）**催化剂利用率显著提高**
- 活性梯度分布与异丁烷浓度分布匹配，上层高活性催化剂快速转化高浓度异丁烷，下层低活性催化剂避免过度反应；
- 催化剂利用率提高20-30%，相同催化剂装填量下，转化率提高5-8个百分点。

（2）**温度控制精度显著提高**
- 分段梯度加热，各段独立控温，温度控制精度从±10-15℃提高至±3℃；
- 避免入口段过冷（影响转化率）和出口段过热（促进副反应），选择性提高3-5个百分点。

（3）**催化剂寿命显著延长**
- 入口段温度不再过热，出口段温度不再欠热，催化剂失活速率降低；
- 催化剂运行周期从1.5年延长至2.5-3年，延长67-100%；
- 催化剂更换成本降低30-40%。

（4）**加热能耗显著降低**
- 分段按需加热，避免过度加热，加热能耗降低15-25%；
- 综合运行成本降低10-20%。

（5）**设备投资显著降低**
- 相比移动床（Oleflex）投资降低40%，相比流化床（FBD-4）投资降低60%以上；
- 设备投资回收期缩短至1-2年。

（6）**操作维护显著简化**
- 无需复杂的催化剂循环和连续再生系统（vs移动床）；
- 无需流态化控制系统（vs流化床）；
- 操作人员培训周期短，维护费用低。

## 具体实施方式

下面结合附图和具体实施例对本发明作进一步详细说明。

### 实施例1

如图1所示，本实施例提供一种用于异丁烷脱氢制异丁烯的多段式反应装置，包括反应器壳体1、活性梯度催化剂床层2、分段梯度加热系统3、支撑结构4、绝热层5、进料分布器6、出气收集器7和智能温控系统8。

**反应器壳体1**

所述反应器壳体1为立式圆筒形结构，材质为耐高温合金钢（S31008或Incoloy 800H），内径为2.8m；总高度为12m；壁厚为30mm；设计压力为0.8MPa，设计温度为650℃。

**与现有技术CN103232312A对比：**该专利的反应器结构与本发明相似，均为固定床结构，但本发明的内部催化剂床层和加热系统不同。

**活性梯度催化剂床层2**

如图2所示，所述活性梯度催化剂床层2沿反应器轴向从上至下依次设置于反应器壳体1内，包括上层床层21、中层床层22和下层床层23。

**与现有技术CN103232312A的关键区别：**

| 对比项目 | CN103232312A | 本发明 | 区别 |
|---------|--------------|--------|------|
| 床层结构 | 单一均一床层 | 三层活性梯度床层 | 结构不同 |
| 上层催化剂 | Pt 0.3%（全床层相同） | Pt 0.4%（高活性） | 活性更高 |
| 中层催化剂 | Pt 0.3%（全床层相同） | Pt 0.25%（中活性） | 活性适中 |
| 下层催化剂 | Pt 0.3%（全床层相同） | Pt 0.15%（低活性） | 活性更低 |
| 助剂K | 无或少量 | 适量添加（抗积碳） | 寿命更长 |

所述上层床层21高度为2.2m；装填高活性脱氢催化剂，装填量为2.5-3.5吨。所述高活性脱氢催化剂为Pt-Sn/Al2O3催化剂，包括载体和负载于载体上的活性组分。所述载体为γ-Al2O3，比表面积为180-220 m²/g，孔容为0.5-0.7 mL/g。所述活性组分包括Pt 0.4wt%、Sn 0.8wt%、K 0.1wt%。

所述中层床层22高度为4.5m；装填中活性脱氢催化剂，装填量为5.0-6.5吨。所述中活性脱氢催化剂为Pt-Sn-K/Al2O3催化剂。所述活性组分包括Pt 0.25wt%、Sn 0.6wt%、K 0.2wt%。

所述下层床层23高度为3.5m；装填低活性脱氢催化剂，装填量为3.5-4.5吨。所述低活性脱氢催化剂为Pt-Sn-K/Al2O3催化剂。所述活性组分包括Pt 0.15wt%、Sn 0.5wt%、K 0.4wt%。

所述上层床层21、中层床层22和下层床层23的床层高度比为1:2.05:1.59，接近理想比例1:2:1.8。

**活性梯度分布原理：**

异丁烷浓度沿床层分布：入口95-98% → 中段60-80% → 出口40-60%

本发明的催化剂活性分布：入口高活性（Pt 0.4%） → 中段中活性（Pt 0.25%） → 出口低活性（Pt 0.15%）

活性分布与浓度分布匹配，使各段催化剂负荷均衡，避免入口过载、出口欠载。

**分段梯度加热系统3**

如图3所示，所述分段梯度加热系统3包括沿反应器轴向从上至下依次设置的第一独立加热单元31、第二独立加热单元32、第三独立加热单元33和第四独立加热单元34。

**与现有技术CN103232312A的关键区别：**

| 对比项目 | CN103232312A | 本发明 | 区别 |
|---------|--------------|--------|------|
| 加热方式 | 外部整体加热 | 四段独立加热 | 控制方式不同 |
| 温度分布 | 整体均匀 | 梯度分布（610→560℃） | 温度分布不同 |
| 温控精度 | ±10-12℃ | ±3℃ | 精度更高 |
| 供热匹配 | 未匹配吸热分布 | 匹配吸热分布 | 效率更高 |

所述第一独立加热单元31对应上层床层21，加热温度为610℃，加热功率为450-550 kW。

所述第二独立加热单元32对应中层床层22上部，加热温度为595℃，加热功率为400-500 kW。

所述第三独立加热单元33对应中层床层22下部，加热温度为580℃，加热功率为350-450 kW。

所述第四独立加热单元34对应下层床层23，加热温度为560℃，加热功率为300-400 kW。

**分段梯度加热原理：**

脱氢反应吸热速率分布：入口高 → 中段中 → 出口低

本发明的供热温度分布：入口610℃ → 中段595/580℃ → 出口560℃

供热分布与吸热分布匹配，使各段温度稳定，避免入口过冷、出口过热。

**智能温控系统8**

所述智能温控系统8包括中央控制器81、温度采集模块82、加热功率控制模块83和数据显示模块84。

所述中央控制器81采用PLC或DCS控制系统，根据预设的温度设定值和采集的温度数据，通过PID控制算法计算各独立加热单元的加热功率调节量。
