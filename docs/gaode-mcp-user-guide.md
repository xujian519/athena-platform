# 🗺️ 高德地图MCP公共服务用户指南

## 📋 概述

高德地图MCP服务是Athena工作平台提供的公共地理空间智能工具，基于高德地图API为AI助手提供完整的地理位置服务能力。所有Athena平台用户都可以通过MCP协议直接使用这些强大的地理功能。

### 🎯 核心功能

- **地理编码**: 地址↔坐标双向转换
- **POI搜索**: 兴趣点智能搜索
- **路径规划**: 多种出行方式导航
- **地图服务**: 天气、静态地图、坐标转换
- **交通服务**: 实时路况查询
- **地理围栏**: 位置监控管理（需企业权限）

## 🚀 快速开始

### 1. AI客户端配置

在您的AI客户端（如Claude Desktop、Cursor等）中添加以下MCP配置：

```json
{
  "mcpServers": {
    "gaode-maps": {
      "command": "python3",
      "args": ["/Users/xujian/Athena工作平台/amap-mcp-server/run_server.py"],
      "cwd": "/Users/xujian/Athena工作平台/amap-mcp-server",
      "env": {
        "AMAP_API_KEY": "4c98d375577d64cfce0d4d0dfee25fb9",
        "MCP_LOG_LEVEL": "INFO"
      }
    }
  }
}
```

### 2. 启动服务

```bash
# 验证MCP服务配置
./dev/scripts/start_public_amap_mcp.sh start

# 检查服务健康状态
python3 dev/scripts/mcp_health_check.py
```

### 3. 开始使用

配置完成后，重启您的AI客户端，就可以直接使用高德地图功能了！

## 🛠️ 工具详细说明

### 1. 📍 地理编码工具 (`gaode_geocode`)

**功能**: 提供地址和坐标之间的双向转换

**用途**:
- 地址标准化处理
- 地图标注定位
- 位置数据 enrichment

**使用示例**:
```
请将"北京市海淀区中关村大街1号"转换为经纬度坐标
请告诉我经纬度116.397455,39.909187对应的详细地址
```

### 2. 🔍 POI搜索工具 (`gaode_poi_search`)

**功能**: 强大的兴趣点搜索功能

**用途**:
- 商家位置查找
- 服务网点分析
- 竞争对手调研
- 区域商业调查

**使用示例**:
```
帮我搜索北京中关村附近的咖啡厅
在天安门周围1公里内找找银行
在朝阳区搜集中餐厅
```

### 3. 🛣️ 路径规划工具 (`gaode_route_planning`)

**功能**: 多种出行方式的智能路径规划

**用途**:
- 出行路线规划
- 物流配送优化
- 服务半径分析
- 可达性分析

**使用示例**:
```
帮我规划从天安门到北京西站的驾车路线
计算从北京南站到首都机场的最快公交路线
从家到公司步行需要多长时间
```

### 4. 🗺️ 地图服务工具 (`gaode_map_service`)

**功能**: 丰富的地图相关服务

**用途**:
- 获取实时天气信息
- 生成静态地图图片
- 坐标系统转换
- 行政区划查询

**使用示例**:
```
查询北京今天的天气情况
将GPS坐标116.407526,39.90403转换为高德坐标
获取上海行政区划信息
为我的报告生成一张北京地图图片
```

### 5. 🚦 交通服务工具 (`gaode_traffic_service`)

**功能**: 实时交通信息查询和分析

**用途**:
- 实时路况查询
- 拥堵路段分析
- 出行时间预测
- 交通规划支持

**使用示例**:
```
查看天安门周围2公里的实时路况
分析长安街的交通状况
帮我判断现在走三环堵不堵车
```

### 6. 🔒 地理围栏工具 (`gaode_geofence`)

**功能**: 地理围栏管理（需要企业级API权限）

**用途**:
- 电子围栏监控
- 区域权限管理
- 位置触发服务
- 地理告警系统

**使用示例**:
```
创建一个以公司为中心、半径1公里的地理围栏
查询指定区域内的所有围栏
```

*注意: 此功能需要高德地图企业级API权限*

## 💡 使用技巧

### 1. 高效查询建议

- **使用具体地址**: 提供详细的地址信息可以提高准确性
- **坐标格式**: 使用"经度,纬度"格式，如"116.397455,39.909187"
- **区域限定**: 在搜索POI时指定城市或区域，提高搜索精度

### 2. 路径规划策略

- **驾车**: 支持避开拥堵、避开收费等策略
- **步行**: 适合短距离出行，会考虑人行道等因素
- **公交**: 提供换乘方案，包括地铁和公交线路
- **骑行**: 适合环保出行，考虑自行车道

### 3. 批量操作

- 地理编码工具支持批量地址转换
- POI搜索支持多边形区域搜索
- 可以通过编程方式批量处理地理位置数据

## 🔧 故障排除

### 常见问题

**Q: MCP工具无法连接**
A: 检查网络连接和API密钥配置，运行健康检查脚本诊断问题

**Q: 某些功能返回权限错误**
A: 地理围栏等功能需要企业级API权限，请联系管理员升级

**Q: 搜索结果不准确**
A: 尝试使用更具体的地址描述，或者先进行地理编码获取精确坐标

**Q: 路径规划返回错误**
A: 确保起点和终点坐标格式正确，使用"经度,纬度"格式

### 健康检查

```bash
# 运行单次健康检查
python3 dev/scripts/mcp_health_check.py --single

# 连续监控服务状态
python3 dev/scripts/mcp_health_check.py --continuous 3 --interval 30

# 保存检查结果
python3 dev/scripts/mcp_health_check.py --save
```

## 📊 性能和限制

### API限制

- **请求频率**: 每分钟最多100次请求
- **日请求量**: 每日最多30,000次请求
- **并发连接**: 最多10个并发请求

### 最佳实践

1. **合理使用**: 避免频繁的重复请求
2. **缓存结果**: 对于不常变化的数据可以进行本地缓存
3. **批量处理**: 尽量将多个操作合并为单次请求
4. **错误处理**: 正确处理API返回的错误信息
