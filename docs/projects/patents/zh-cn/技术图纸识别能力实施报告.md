## 🔧 核心功能详解

### 1. 智能分块处理

**功能**: 自动将大图纸(>1024x1024)分成多个小块,逐块分析后合并结果

**实现**:
```python
async def _chunk_image(self, image_data: bytes) -> List[ImageChunk]:
    """将图像分块"""
    img = Image.open(BytesIO(image_data))
    width, height = img.size

    chunks = []
    chunk_id = 0

    # 计算分块数量
    cols = (width + self.chunk_size - 1) // self.chunk_size
    rows = (height + self.chunk_size - 1) // self.chunk_size

    for row in range(rows):
        for col in range(cols):
            # 裁剪、编码、保存
            ...
```

**优势**:
- ✅ 解决上下文窗口限制
- ✅ 提高细节识别准确度
- ✅ 支持任意大小图纸

### 2. 上下文压缩集成

**功能**: 自动压缩对话历史,避免超出token限制

**实现**:
```python
self.context_compressor = ContextCompressor(
    max_history_length=3,  # 只保留最近3轮对话
    max_tokens=6000,       # 为图像留出空间
    compression_strategy="importance"  # 重要性压缩
)
```

**优势**:
- ✅ 自动管理token预算
- ✅ 保留重要信息
- ✅ 提高请求成功率

### 3. 三级分析深度

| 级别 | Token消耗 | 输出内容 | 适用场景 |
|------|----------|----------|----------|
| **BASIC** | ~500 | 类型+标题+描述 | 快速查看 |
| **INTERMEDIATE** | ~1500 | +组件+特征 | 标准分析 |
| **ADVANCED** | ~3000 | +原理+关联 | 深度理解 |

### 预期性能

| 图纸尺寸 | 处理方式 | 预期耗时 | Token消耗 |
