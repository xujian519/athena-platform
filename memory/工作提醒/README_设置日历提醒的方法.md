## 方法二：设置提醒事项

1. 打开提醒事项应用
2. 新建列表：工作
3. 添加提醒：
   - 任务：爸爸的工作任务
   - 日期：2024年12月15日
   - 时间：上午 9:00
   - 附注：同上备注内容

## 方法三：手机快捷设置

1. 打开提醒事项或日历
2. 点击"+"创建新提醒
3. 设置时间和内容
