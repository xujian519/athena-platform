## 🎯 核心功能

### 1. 请求去重

**原理**: 使用MD5哈希识别重复向量

**收益**: 
- 重复请求节省100%计算
- 减少70%网络往返

**实现**:
```go
cacheKey := MD5(collection + vector[0:10])
```

### 2. 智能分组

**原理**: 按集合名称和优先级分组

**收益**:
- 提高缓存命中率
- 优化资源利用

**实现**:
```go
maxGroupSize := 50
priority = req.Limit * 10 + req.Threshold * 5
```

### 3. 请求合并

**原理**: 相似向量（>95%相似度）自动合并

**收益**:
- 相似请求减少50% API调用
- 批量效率提升30%

**实现**:
```go
similarity = dotProduct(vec1, vec2) / (norm(vec1) * norm(vec2))
if similarity >= 0.95 {
    mergedReq.Limit *= len(similarVectors)
}
```

### 4. 动态批量大小

**原理**: 自适应批量大小

**收益**:
- 小批量: 不拆分，低延迟
- 大批量: 拆分，高吞吐

**实现**:
```go
optimalSize = sqrt(requestCount) * 10
```
