## 【CAP01: 法律检索】

### 流程
```
提取关键词 → 向量检索(top-5) → 知识图谱查询 → 调用法条画像 → 综合排序
```

### 输出格式
```markdown
【检索结果】(共X条)

## 法条类 (X条)
1. 《专利法》第XX条第X款 (相似度: XX%, 引用: XXX次)
   - 条文内容: [内容]
   - 效力层级: [层级]
   - 应用画像: 引用频次、趋势、场景、倾向
   - 关系画像: 上位法、配套、关联法条
   - 实务画像: 典型案例、争议焦点、支持率

## 案例类 (X条)
1. 决定书#XXXXXX (日期, 相似度: XX%)
   - 案由: [案由]
   - 争议焦点: [焦点]
   - 审查员观点: [观点]
   - 裁判要旨: [要旨]
   - 适用场景: [场景]
```

```markdown
技术问题: [问题]
技术进步: [进步]
```

```markdown
   [同上结构]
```

## 【CAP05: 创造性分析】

### 三步法
```
(1) 最接近现有技术 → 技术领域/问题/效果相似
(2) 区别特征 → 实际解决的技术问题
(3) 技术启示 → 是否显而易见
```

### 输出格式
```markdown
【创造性分析】
最接近现有技术: [D1]
区别特征: [特征]
技术问题: [问题]
技术启示: [D2是否给出启示]
结论: [具备/不具备创造性]
```
