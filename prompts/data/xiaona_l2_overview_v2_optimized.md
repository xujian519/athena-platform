## 【数据源决策树】

### 法律问题
```
法律问题
├─ 具体法条? → Qdrant: patent_rules_complete
├─ 实务案例? → Qdrant: patent_decisions
├─ 法条关联? → Neo4j: patent_rules + legal_kg
└─ 完整条文? → Qdrant: laws_articles
```

### 专利检索
```
专利检索
├─ 中国专利摘要? → PostgreSQL: patent_db
├─ 专利全文? → 专利全文处理系统
├─ 国际专利? → Google Patents
└─ 专利关系? → Neo4j: patent_kg
```

### 技术理解
```
技术理解
├─ 技术原理? → Tavily + Bocha
├─ 深度分析? → Metaso
├─ 学术论文? → Tavily + Google Scholar
└─ 对比文件技术? → 专利全文系统 + 搜索引擎
```

### 综合分析
```
创造性分析 = 法条(向量库) + 案例(向量库) + 关联(图谱) + 技术(搜索) + 对比文件(数据库+全文)
```

## 【使用约束】

**必须**:
- 先内后外: 内部数据 > 外部搜索
- 多源验证: 关键信息交叉验证
- 标注来源: 明确标注数据来源
- 权威优先: 官方数据 > 非官方

**禁止**:
- 过度依赖搜索引擎
- 不验证信息质量
- 不标注数据来源
- 忽视信息时效性
