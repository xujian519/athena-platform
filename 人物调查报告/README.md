## 📝 报告类型说明

### 1. 律师背景调查报告
**包含内容**:
- 基本信息（姓名、性别、联系方式）
- 职业信息（律所、职务、执业年限）
- 专业领域（知识产权、公司法律、破产重整等）
- 职业经历（执业历史、律所变迁）
- 社会任职（行业协会职务、学术活动）
- 案件经验（典型案例、胜诉率）
- 核心优势（专业特长、资源优势）
- 适用场景（推荐合作场景）

**文件命名格式**: `[姓名]律师背景调查报告_[YYYYMMDD].md`

### 2. 专利代理人能力评估报告
**包含内容**:
- 基本信息与资质认证
- 技术领域专长
- 专利撰写经验
- OA答复能力
- 无效宣告经验
- 诉讼代理经验
- 客户评价

**文件命名格式**: `[姓名]专利代理人能力评估_[YYYYMMDD].md`

### 3. 审查员分析报告
**包含内容**:
- 审查员基本信息
- 所属技术领域
- 审查倾向分析
- 历史审查案例
- 沟通风格特点

**文件命名格式**: `[姓名]审查员分析报告_[YYYYMMDD].md`

### 4. 客户背景调查报告
**包含内容**:
- 企业/个人基本信息
- 业务领域与产品线
- 知识产权现状
- 合作历史与评价
- 风险评估

**文件命名格式**: `[客户名称]背景调查报告_[YYYYMMDD].md`

## 🔧 使用指南

### 添加新报告

#### 步骤1: 确定分类
根据人物类型选择对应的子目录：
- 律师 → `律师/`
- 专利代理人 → `专利代理人/`
- 审查员 → `审查员/`
- 客户 → `客户/`
- 其他 → `其他/`

#### 步骤2: 创建报告文件
按照标准格式创建调查报告文件，文件命名格式：
```
[姓名]_[报告类型]_[YYYYMMDD].md
```

示例：
```
张三专利代理人能力评估_20260109.md
李四律师背景调查报告_20260109.md
```

#### 步骤3: 更新索引数据库
在 `联系人数据库.md` 中添加新人物的索引信息：

```markdown
### [姓名] - [职业/专长]

**基本信息**:
- **姓名**: [姓名]
- **性别**: [性别]
- **手机**: [手机号]
- **邮箱**: [邮箱]
- **办公地址**: [地址]

**职业信息**:
- **单位**: [公司/机构名称]
- **职务**: [职位]
- **专业领域**: [专业领域]

**调查报告**: [报告文件名]
**添加日期**: [日期]
**最后更新**: [日期]
**备注**: [备注信息]
```

#### 步骤4: 更新统计
在本README的"当前统计"部分更新数字。

### 查找报告

#### 按类别查找
1. 进入对应的子目录
2. 浏览该类别下的所有报告

#### 按姓名查找
1. 打开 `联系人数据库.md`
2. 使用Ctrl+F搜索姓名
3. 找到对应的报告文件路径
