## 🚀 快速开始

### 第一步：配置邮箱信息

首次运行会自动生成配置文件模板，位置：
```
production/scripts/client_development_data/email_config.json
```

**配置内容**：
```json
{
  "smtp_server": "smtp.qq.com",        // SMTP服务器（QQ邮箱）
  "smtp_port": 587,                    // SMTP端口
  "sender_email": "your@email.com",    // 👈 需要填写：您的邮箱
  "sender_password": "your_password",  // 👈 需要填写：邮箱授权码
  "sender_name": "徐健",               // 👈 需要填写：您的姓名
  "sender_phone": "138-xxxx-xxxx",     // 👈 需要填写：您的电话
  "sender_wechat": "your_wechat_id"    // 👈 需要填写：您的微信号
}
```

### 第二步：获取邮箱授权码

**QQ邮箱获取授权码步骤**：
1. 登录QQ邮箱网页版
2. 点击"设置" → "账户"
3. 找到"POP3/IMAP/SMTP/Exchange/CardDAV/CalDAV服务"
4. 开启"IMAP/SMTP服务"
5. 按照提示发送短信验证
6. 获得16位授权码（类似：xxxxxxxxxxxxxxxx）
7. 将授权码填入配置文件的 `sender_password` 字段

**其他邮箱SMTP配置**：

| 邮箱服务商 | SMTP服务器 | 端口 |
|-----------|-----------|------|
| QQ邮箱 | smtp.qq.com | 587 |
| 163邮箱 | smtp.163.com | 465 |
| Gmail | smtp.gmail.com | 587 |
| 企业微信 | smtp.exmail.qq.com | 465 |

### 第三步：启动系统

```bash
# 进入脚本目录
cd /Users/xujian/Athena工作平台/production/scripts

# 运行自动化邮件系统
python auto_email_client_development.py
```

## 📋 系统功能说明

启动后会显示交互菜单：

```
============================================================
请选择操作：
1. 立即发送第一周邮件
2. 查看发送统计报告
3. 设置每周定时发送
4. 手动发送指定周的邮件
5. 退出
============================================================
```

### 选项1：立即发送第一周邮件
- 向所有目标客户发送第一周的开发邮件
- 每封邮件间隔30秒，避免被识别为垃圾邮件
- 自动记录发送历史

### 选项2：查看发送统计报告
显示详细的发送统计：
- 总发送数量和成功率
- 各公司的发送详情
- 失败原因分析
- 改进建议

### 选项3：设置每周定时发送
- 设置定时任务（默认：每周二上午10:00）
- 系统持续运行，自动发送
- 按Ctrl+C可退出

### 选项4：手动发送指定周的邮件
- 选择要发送的周数（1-N）
- 灵活控制发送进度
- 适合特殊时机发送

## 🔧 故障排查

### 问题1：邮件发送失败
**可能原因**：
- 邮箱密码或授权码错误
- SMTP服务器配置错误
- 网络连接问题

**解决方法**：
- 检查配置文件中的邮箱信息是否正确
- 确认已开启SMTP服务
- 尝试使用其他网络

### 问题2：被识别为垃圾邮件
**解决方法**：
- 增加邮件发送间隔（改为60秒）
- 使用企业邮箱而非个人邮箱
- 在邮件内容中增加更多个性化信息

### 问题3：定时任务不执行
**解决方法**：
- 确保程序持续运行
- 使用systemd或cron配置自启动
- 检查系统时间是否正确
