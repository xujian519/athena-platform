## 🚀 快速部署指南

### 5分钟部署

```bash
# 1. 设置环境
export ATHENA_PROJECT_ROOT="/path/to/Athena工作平台"
export PYTHONPATH="$ATHENA_PROJECT_ROOT:$PYTHONPATH"

# 2. 验证安装
cd $ATHENA_PROJECT_ROOT
python3 -m pytest tests/test_ethics/test_constitution.py -q

# 3. 启动监控
python3 -c "
from core.ethics.monitoring_prometheus import setup_prometheus_monitoring
monitor = setup_prometheus_monitoring(port=9091)
print('✅ 监控已启动')
"

# 4. 测试评估
python3 -c "
from core.ethics import create_ethics_evaluator
evaluator = create_ethics_evaluator()
result = evaluator.evaluate_action(
    agent_id='test',
    action='测试行动',
    context={'confidence': 0.9}
)
print(f'状态: {result.status.value}')
print(f'评分: {result.overall_score}')
"

# 5. 验证metrics
curl http://localhost:9091/metrics | grep ethics_evaluation_total
```
